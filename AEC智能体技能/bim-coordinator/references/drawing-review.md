# 各专业图纸审查重点

## 建筑专业

- [ ] 建筑面积计算是否符合规范
- [ ] 防火分区面积是否超限，防火门/防火卷帘位置
- [ ] 疏散楼梯数量、宽度、距离是否满足规范
- [ ] 无障碍设计是否完整
- [ ] 建筑节能设计（外墙/屋面/门窗热工参数）
- [ ] 卫生间、厨房防水范围
- [ ] 门窗表与平面图是否一致
- [ ] 室内外高差、散水坡度

## 结构专业

- [ ] 结构体系与建筑方案是否匹配
- [ ] 基础形式与地质报告是否对应
- [ ] 构件截面尺寸与建筑净高是否协调
- [ ] 预留洞口位置与机电是否一致
- [ ] 后浇带位置与施工缝设置
- [ ] 特殊节点（转换层/悬挑/连体）详图是否完整
- [ ] 钢筋锚固、搭接长度标注
- [ ] 混凝土强度等级标注是否完整

## 给排水专业

- [ ] 给水系统分区是否合理（每区≤50m）
- [ ] 消防系统（喷淋/消火栓）覆盖是否完整
- [ ] 排水坡度标注（污水≥0.5%，雨水≥0.3%）
- [ ] 管道穿墙/楼板套管设置
- [ ] 水泵房、水箱间位置与结构是否协调
- [ ] 卫生间排水立管位置与建筑是否冲突
- [ ] 室外管网与市政接口

## 暖通专业

- [ ] 风管截面尺寸与建筑净高是否协调
- [ ] 机房（空调机房/风机房）面积是否足够
- [ ] 防排烟系统（加压送风/机械排烟）覆盖
- [ ] 风口位置与建筑装修是否协调
- [ ] 冷热源系统（冷机/锅炉）容量计算
- [ ] 管道穿防火墙防火阀设置

## 电气专业

- [ ] 变配电室位置与负荷中心是否合理
- [ ] 配电系统图与平面图是否一致
- [ ] 消防电源（双路/EPS）配置
- [ ] 防雷接地系统（接地电阻要求）
- [ ] 桥架路由与其他专业是否冲突
- [ ] 应急照明、疏散指示灯设置
- [ ] 弱电系统（综合布线/安防/楼控）点位

## 专业间协调重点

| 协调内容 | 涉及专业 | 常见问题 |
|---------|---------|---------|
| 梁底净高 | 结构+机电 | 管线穿梁或净高不足 |
| 卫生间降板 | 建筑+结构+给排水 | 降板深度不一致 |
| 设备基础 | 结构+机电 | 基础位置/荷载未提资 |
| 预留洞口 | 结构+机电 | 洞口位置/尺寸不一致 |
| 屋面设备 | 建筑+结构+机电 | 荷载未考虑/防水收口 |

# FAQ

## 常见问题

### Q1: BIM模型精度 LOD 如何选择？

**A**: 根据项目阶段和应用目标选择：
- 方案阶段：LOD 100-200
- 初步设计：LOD 200-300
- 施工图设计：LOD 300-350
- 施工阶段：LOD 350-400
- 竣工移交：LOD 500

### Q2: 多个专业模型如何高效协同？

**A**: 建议采用以下方式：
1. 建立协同平台（如BIM360、ArchiCAD）
2. 设置统一的项目基点和坐标
3. 制定工作集划分规则
4. 建立定期同步机制
5. 明确模型访问权限

### Q3: 碰撞检查发现的问题如何高效处理？

**A**: 处理流程：
1. 按严重程度分类（一级：影响结构安全；二级：影响功能使用；三级：影响观感）
2. 通过协同平台派发任务给责任专业
3. 整改后进行复验
4. 问题关闭后归档
5. 定期统计问题数量和类型

### Q4: BIM模型如何进行版本管理？

**A**: 版本管理要点：
1. 命名规则：[专业]_[区域]_[版本号]_[日期]
2. 每周定期备份
3. 重大版本节点保存
4. 建立版本对比机制
5. 保留历史版本备查

### Q5: 施工阶段BIM模型如何维护？

**A**: 维护措施：
1. 指定专人负责模型更新
2. 建立变更通知机制
3. 现场问题及时反馈到模型
4. 定期进行模型与现场核对
5. 模型与施工进度同步

### Q6: BIM成果如何交付？

**A**: 交付内容：
1. 完整BIM模型文件（.rvt/.rfa格式）
2. 碰撞检查报告
3. 管线综合图纸
4. 净高分析报告
5. 施工模拟视频
6. BIM应用总结报告

### Q7: BIM团队人员如何配置？

**A**: 常规配置：
- 小型项目（＜2万㎡）：1-2人
- 中型项目（2-10万㎡）：3-5人
- 大型项目（10万㎡以上）：5-10人
- 超大项目：按专业配置，每专业2-3人

## 疑难解答

| 问题 | 解决方案 |
|------|----------|
| 模型文件过大打开缓慢 | 进行模型轻量化，分区域、分层加载 |
| 各专业模型坐标不一致 | 统一设置项目基点和共享坐标原点 |
| 碰撞问题反复出现 | 建立问题追踪机制，加强设计阶段审核 |
| 模型无法导出算量 | 检查构件属性是否完整，映射规则是否正确 |
| 现场不按模型施工 | 加强BIM交底，建立模型变更反馈机制 |

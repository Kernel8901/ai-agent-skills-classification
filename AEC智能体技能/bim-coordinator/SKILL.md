# BIM应用与图纸会审助手

## 图纸会审

### 会审流程
```
图纸下发 → 各专业自审（7天）→ 汇总问题清单
→ 会审会议（建设/设计/施工/监理）
→ 问题解答与设计变更 → 会审纪要签发
```

### 图纸问题分类

| 类别 | 说明 | 处理方式 |
|------|------|---------|
| A类-设计错误 | 计算错误、违反规范 | 设计出变更图 |
| B类-图纸矛盾 | 平立剖不一致、专业间矛盾 | 设计书面确认 |
| C类-深度不足 | 节点不清、尺寸缺失 | 设计补充说明 |
| D类-施工建议 | 可优化的施工方案 | 设计确认后执行 |

### 图纸会审问题清单格式
```
| 序号 | 图纸编号 | 问题位置 | 问题描述 | 问题类别 | 提出单位 | 设计回复 | 处理结果 |
|------|---------|---------|---------|---------|---------|---------|---------|
```

## BIM应用策划

### LOD等级定义

| LOD等级 | 应用阶段 | 模型精度 |
|---------|---------|---------|
| LOD 100 | 方案设计 | 体量模型 |
| LOD 200 | 初步设计 | 近似几何 |
| LOD 300 | 施工图设计 | 精确几何 |
| LOD 350 | 施工深化 | 含连接件 |
| LOD 400 | 施工安装 | 制造级精度 |
| LOD 500 | 竣工交付 | 竣工实际状态 |

### 施工阶段BIM应用点

**高价值应用（优先实施）**：
- 碰撞检查（机电综合管线）
- 施工方案模拟（复杂节点/危大工程）
- 工程量提取与复核
- 4D进度模拟

**扩展应用**：
- 场地布置模拟
- 预制构件深化
- 竣工模型交付

## 碰撞检查报告格式

```
碰撞检查报告

项目名称：        检查日期：
检查软件：        检查范围：

一、碰撞统计
| 碰撞类型 | 数量 | 严重程度 |
|---------|------|---------|
| 硬碰撞（实体重叠） | XX | 高 |
| 软碰撞（间距不足） | XX | 中 |

二、主要碰撞问题
| 编号 | 碰撞位置 | 碰撞专业 | 问题描述 | 建议解决方案 | 责任方 |
|------|---------|---------|---------|------------|--------|

三、处理建议
优先处理：[影响主体结构/消防/主干管道的碰撞]
协调原则：[小管让大管/有压让无压/电气让水暖]
```

## 机电综合管线排布原则

```
优先级（从高到低）：
1. 消防管道（不可移动）
2. 重力排水管（坡度要求）
3. 通风管道（截面大，优先排布）
4. 给水/空调水管
5. 电缆桥架
6. 弱电桥架

净高控制：
- 走廊：≥2.2m（公共建筑）
- 机房：≥2.5m
- 地下车库：≥2.2m（行车道≥2.4m）
```

# Calculation Guide

## 结构计算基本公式

### 1. 混凝土结构

#### 1.1 正截面承载力计算
**公式**: M ≤ fy × As × (h0 - As' × fy / (α1 × fc × b))

其中：
- M: 弯矩设计值
- fy: 钢筋抗拉强度设计值
- As: 纵向受拉钢筋面积
- h0: 截面有效高度
- α1: 系数（C50以下取1.0，C80取0.94）
- fc: 混凝土轴心抗压强度设计值
- b: 截面宽度

**简化计算**（单筋矩形截面）：
```
已知：b=200mm, h=500mm, h0=465mm, C30混凝土, HRB400钢筋
fc=14.3N/mm², fy=360N/mm²
求：受拉钢筋面积As
解：α1×fc×b×h0²/M = 1×14.3×200×465²/100000000 ≈ 0.35
查表得：ξ=0.45, As=M/(fy×γs×h0) = 100000000/(360×0.775×465) = 775mm²
```

#### 1.2 斜截面承载力
**公式**: V ≤ 0.7×ft×b×h0 + fyv×Asv×h0/s

其中：
- V: 剪力设计值
- ft: 混凝土轴心抗拉强度设计值
- fyv: 箍筋抗拉强度设计值
- Asv: 箍筋面积
- s: 箍筋间距

### 2. 钢结构

#### 2.1 稳定性计算
**公式**: N/φA ≤ f

其中：
- N: 轴力设计值
- φ: 稳定系数（根据长细比查表）
- A: 截面面积
- f: 钢材抗压强度设计值

**长细比计算**：λ = l0 / i
- l0: 计算长度
- i: 回转半径

#### 2.2 强度计算
**公式**: σ = N/A + M/W ≤ f

其中：
- N: 轴力
- M: 弯矩
- A: 截面面积
- W: 截面模量

---

### 3. 基础计算

#### 3.1 地基承载力验算
**公式**: p ≤ fa

其中：
- p: 基底平均压力
- fa: 地基承载力特征值

**基底压力计算**：
p = (F + G) / A
- F: 上部结构荷载
- G: 基础自重
- A: 基础底面积

#### 3.2 基础沉降计算
**公式**: s = ψs × Σ(si)

其中：
- s: 最终沉降量
- ψs: 沉降计算经验系数
- si: 各分层沉降量

## 注意事项

### 计算要点
1. 荷载取值要完整，考虑各种工况
2. 计算模型要符合实际
3. 结果要与规范限值对比
4. 构造措施要满足要求

### 常见问题
1. 计算简图与实际不符
2. 荷载取值遗漏
3. 稳定计算不全面
4. 节点设计不合理

# FAQ

## 常见问题

### Q1: 结构方案如何选择？

**A**：选择结构方案需要考虑：
1. **建筑功能**：不同功能对结构有不同要求
2. **建筑高度**：高度决定结构体系
3. **抗震设防**：根据设防类别选择结构类型
4. **经济性**：综合考虑造价和维护成本
5. **施工条件**：考虑当地施工能力

常用选择：
- 多层住宅：框架结构或框架剪力墙
- 高层住宅：剪力墙结构
- 商业综合体：框架核心筒结构
- 大跨度建筑：钢结构或预应力混凝土

### Q2: 含钢量指标多少合适？

**A**：常规含钢量指标（kg/平米）：
- 多层住宅：35-50
- 高层住宅（30层以下）：45-65
- 高层住宅（30层以上）：55-80
- 商业/办公：55-90
- 地下室：100-150

影响因素：建筑高度、平面布局、设防烈度等

### Q3: 既有建筑如何加固？

**A**：常用加固方法：
1. **混凝土加固**：增大截面、粘贴钢板、碳纤维
2. **钢结构加固**：焊接、螺栓连接、增设构件
3. **基础加固**：注浆、桩基托换、扩大基础

加固前必须进行可靠性鉴定，加固方案要由专业设计。

### Q4: 深基坑如何支护？

**A**：支护方案选择考虑因素：
- 基坑深度
- 地质条件
- 周边环境
- 施工条件
- 经济性

常用支护形式：
- 浅基坑（＜5m）：土钉墙、简单放坡
- 中等基坑（5-10m）：桩锚、桩撑
- 深基坑（＞10m）：灌注桩+锚索/内支撑

### Q5: 钢结构优化从哪些方面入手？

**A**：钢结构优化方向：
1. **结构体系优化**：选择合理结构形式
2. **截面优化**：采用高效截面
3. **节点优化**：简化节点构造
4. **材料优化**：采用高强度钢材
5. **施工优化**：考虑制作安装方便

### Q6: 结构咨询报告包括哪些内容？

**A**：结构咨询报告应包含：
1. 工程概况
2. 设计条件（荷载、地震等）
3. 结构方案分析
4. 计算复核结果
5. 关键问题及建议
6. 技术经济指标
7. 结论和建议

## 疑难解答

| 问题 | 解决方案 |
|------|----------|
| 结构计算不收敛 | 检查模型约束、荷载，调整计算参数 |
| 位移超限 | 加强侧向刚度，优化结构布置 |
| 配筋过大 | 调整截面尺寸，优化结构方案 |
| 节点应力集中 | 局部加强，优化节点构造 |

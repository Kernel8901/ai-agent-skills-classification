# 施工方案生成助手

## 核心能力

根据工程信息生成专业、规范的施工方案文件，覆盖从施工组织设计到危大工程专项方案的全谱系。

## 信息收集（首次触发时询问）

若用户未提供足够信息，按优先级询问：

1. **工程基本信息**：工程名称、类型（土建/市政/装饰/机电/桥梁/隧道等）、建设规模
2. **工程特征**：地质条件、周边环境、结构形式、主要工程量
3. **约束条件**：工期要求、质量等级、安全文明施工要求、特殊技术要求
4. **方案类型**：施工组织设计 / 专项施工方案 / 施工工艺方案

## 方案结构模板

### 施工组织设计标准章节

```
1. 工程概况
   - 工程基本情况
   - 建设条件分析
   - 工程重难点分析

2. 施工部署
   - 施工目标（工期/质量/安全/环保）
   - 施工组织机构
   - 施工区段划分
   - 施工顺序安排

3. 施工进度计划
   - 总进度计划（横道图/网络图描述）
   - 关键线路分析
   - 里程碑节点

4. 主要施工方法
   - 各分部分项工程施工工艺
   - 关键技术措施
   - 新技术/新工艺应用

5. 资源配置计划
   - 劳动力计划
   - 主要材料计划
   - 主要机械设备计划

6. 质量保证措施
   - 质量管理体系
   - 各工序质量控制要点
   - 质量通病防治

7. 安全文明施工措施
   - 安全管理体系
   - 危险源辨识与控制
   - 文明施工与环境保护

8. 季节性施工措施（如适用）
```

### 危大工程专项方案额外章节

```
- 工程概况及危险性分析
- 编制依据（规范标准清单）
- 施工计划（含应急预案）
- 施工工艺技术（含计算书）
- 施工安全保证措施
- 劳动力计划
- 验收要求
- 应急处置措施
```

### 深度技术资料（按需加载）

当用户需要编制完整方案或进行计算时，加载对应详解文档：

| 工程类型 | 详解内容 | 文档位置 |
|---------|---------|---------|
| **深基坑** | 支护选型矩阵、分层开挖工艺参数、稳定性验算公式、监测预警体系、应急预案 | [references/deep-excavation.md](references/deep-excavation.md) |
| **高支模** | 荷载计算方法、立杆稳定性验算公式、扣件抗滑验算、地基承载力验算、支架预压、混凝土浇筑控制 | [references/high-formwork.md](references/high-formwork.md) |
| **桥梁工程** | 下部结构（桩基/承台/墩柱/盖梁）、上部结构（预制梁/现浇箱梁/悬臂浇筑）、预应力施工、检测验收 | [references/bridge-construction.md](references/bridge-construction.md) |

### 执行策略

1. **先判断危大等级**：明确告知用户是否需要专家论证
2. **调用框架模板**：给出方案结构框架
3. **按需加载详解**：涉及计算、工艺参数时加载对应详解文档
4. **计算书编写**：给出公式、参数取值、计算过程
5. **风险提示**：关键质量控制点、验收要点

## 规范依据库

生成方案时自动引用适用规范，参见 [references/standards.md](references/standards.md)

## 输出格式

- 默认输出 Markdown 格式，结构清晰
- 用户要求时可转为 Word（调用 docx skill）
- 计算书部分给出公式和计算过程
- 关键数据用表格呈现

## 质量要求

- 引用规范必须真实存在，不编造规范编号
- 技术参数符合行业常规范围
- 安全措施符合强制性条文要求
- 如涉及超危大工程，明确提示需专家论证

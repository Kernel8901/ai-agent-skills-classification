# 危大工程专项方案模板库

## 一、深基坑专项施工方案

### 适用范围
- 开挖深度超过5m（含5m）的基坑
- 开挖深度虽未超过5m，但地质条件、周围环境和地下管线复杂的基坑

### 方案框架

```
1. 工程概况
   1.1 工程基本情况（建设规模、基坑尺寸、开挖深度）
   1.2 工程地质条件（地层描述、地下水位、土层参数）
   1.3 周边环境（建筑物、管线、道路距离）
   1.4 工程重难点分析

2. 编制依据
   - GB 50007-2011 建筑地基基础设计规范
   - JGJ 120-2012 建筑基坑支护技术规程
   - GB 50497-2019 建筑基坑工程监测技术标准
   - 岩土工程勘察报告
   - 基坑支护设计图纸

3. 支护方案
   3.1 支护形式选择（放坡/土钉墙/排桩/地连墙）
   3.2 支护结构设计参数
   3.3 降水方案（降水井布置、降水深度）
   3.4 截水帷幕（如需）

4. 施工计划
   4.1 施工顺序（分层分段开挖）
   4.2 施工进度计划
   4.3 资源配置（机械/人员）

5. 施工工艺技术
   5.1 土方开挖工艺
      - 分层开挖厚度（≤2m/层）
      - 开挖顺序（先中间后两侧/分段跳挖）
      - 坑底保护（预留200-300mm人工清底）
   5.2 支护结构施工工艺
      - 排桩施工（钻孔/旋挖/人工挖孔）
      - 锚索施工（钻孔/注浆/张拉锁定）
      - 土钉施工（成孔/置筋/注浆）
   5.3 降水施工工艺
      - 降水井施工（成孔/下管/填砾/洗井）
      - 降水运行管理

6. 计算书
   6.1 支护结构稳定性验算
      - 整体稳定性（圆弧滑动）
      - 抗倾覆验算
      - 抗隆起验算
      - 抗渗流验算
   6.2 支护结构内力计算
   6.3 锚索/土钉承载力计算

7. 监测方案
   7.1 监测项目
      | 监测项目 | 监测频率 | 预警值 | 报警值 |
      |---------|---------|--------|--------|
      | 支护结构水平位移 | 1次/天 | 30mm | 50mm |
      | 支护结构沉降 | 1次/天 | 20mm | 30mm |
      | 地下水位 | 1次/天 | - | 设计水位+500mm |
      | 周边建筑沉降 | 1次/天 | 20mm | 30mm |
      | 周边地表沉降 | 1次/天 | 30mm | 50mm |
   7.2 监测点布置
   7.3 信息化施工要求

8. 安全保证措施
   8.1 坑边防护（1.2m高防护栏杆+警示标志）
   8.2 坑边荷载控制（距坑边≥1.5m，堆载≤20kPa）
   8.3 排水措施（截水沟+集水坑）
   8.4 应急预案
      - 坑壁坍塌应急处置
      - 突涌水应急处置
      - 周边建筑沉降超限应急处置

9. 文明施工与环境保护
```

### 超危大工程判定（需专家论证）
- 开挖深度超过**8m**
- 开挖深度虽未超过8m，但地质条件或周边环境极其复杂

## 二、高支模专项施工方案

### 适用范围
- 搭设高度**8m及以上**
- 搭设跨度**18m及以上**
- 施工总荷载（设计值）**15kN/m²及以上**
- 集中线荷载（设计值）**20kN/m及以上**

### 方案框架

```
1. 工程概况
   1.1 高支模区域描述（位置/层高/梁板截面）
   1.2 荷载分析（模板自重/混凝土重/施工荷载）

2. 编制依据
   - JGJ 162-2008 建筑施工模板安全技术规范
   - JGJ 130-2011 建筑施工扣件式钢管脚手架安全技术规范
   - GB 50666-2011 混凝土结构工程施工规范

3. 支撑体系设计
   3.1 立杆间距（纵向×横向）
   3.2 步距
   3.3 扫地杆设置（距地面≤200mm）
   3.4 剪刀撑设置（竖向+水平）
   3.5 顶托设置（伸出长度≤300mm）
   3.6 与结构拉结措施

4. 计算书
   4.1 荷载计算
      - 模板及支架自重
      - 新浇混凝土自重（含钢筋）
      - 施工人员及设备荷载（≥2.5kN/m²）
      - 振捣混凝土产生的荷载（≥2kN/m²）
   4.2 立杆稳定性验算
      - 计算长度：l0 = k×μ×h
      - 稳定系数φ查表
      - 验算：N/(φ×A) ≤ [f]
   4.3 扣件抗滑承载力验算
   4.4 地基承载力验算

5. 施工工艺
   5.1 搭设顺序
      立杆→纵横水平杆→剪刀撑→顶托→主次楞→模板
   5.2 搭设要求
      - 立杆垂直度偏差≤H/500且≤50mm
      - 扣件拧紧力矩40-65N·m
   5.3 混凝土浇筑要求
      - 分层浇筑（≤500mm/层）
      - 对称浇筑，避免偏心荷载
      - 浇筑速度控制
   5.4 拆除要求
      - 拆除时间（混凝土强度达到设计值的100%）
      - 拆除顺序（先拆非承重，后拆承重）

6. 验收要求
   - 搭设完成后由项目技术负责人组织验收
   - 验收合格后方可浇筑混凝土
   - 验收记录表（立杆间距/步距/剪刀撑/扣件）

7. 监测方案
   - 浇筑过程中专人监测支撑体系变形
   - 发现异常立即停止浇筑，疏散人员

8. 应急预案
   - 支撑体系失稳应急处置
   - 模板坍塌应急处置
```

### 超危大工程判定（需专家论证）
- 搭设高度**18m及以上**
- 搭设跨度**36m及以上**
- 施工总荷载（设计值）**20kN/m²及以上**
- 集中线荷载（设计值）**30kN/m及以上**

## 三、桥梁工程施工方案

### 适用范围
城市桥梁、公路桥梁、立交桥等各类桥梁工程

### 方案框架

```
1. 工程概况
   1.1 桥梁基本情况（桥型/跨径/宽度/荷载等级）
   1.2 地质水文条件
   1.3 交通组织要求
   1.4 工程重难点

2. 编制依据
   - JTG/T F50-2011 公路桥涵施工技术规范
   - CJJ 2-2008 城市桥梁工程施工与质量验收规范
   - GB 50923-2013 城市轨道交通工程监测技术规范（如适用）

3. 施工总体方案
   3.1 施工顺序（下部结构→上部结构→附属工程）
   3.2 施工分区与流水段划分
   3.3 交通导改方案

4. 下部结构施工
   4.1 桩基础施工
      - 钻孔灌注桩：成孔→清孔→钢筋笼→水下混凝土
      - 预应力管桩：沉桩→接桩→截桩
   4.2 承台施工
      - 基坑开挖与支护
      - 承台模板与混凝土
   4.3 墩柱施工
      - 模板体系（翻模/滑模/爬模）
      - 混凝土浇筑（分层≤300mm）
      - 养护（覆盖保湿≥14天）
   4.4 盖梁施工
      - 支架方案（落地支架/托架）
      - 预应力张拉（如有）

5. 上部结构施工
   5.1 预制梁施工（T梁/箱梁/小箱梁）
      - 预制场布置
      - 模板与钢筋
      - 混凝土浇筑与养护
      - 预应力张拉与压浆
      - 存梁与运输
      - 架梁（架桥机/吊车）
   5.2 现浇箱梁施工
      - 支架方案（满堂支架/贝雷架）
      - 支架预压（消除非弹性变形）
      - 模板安装
      - 钢筋绑扎
      - 混凝土浇筑（对称/分层）
      - 预应力张拉
   5.3 悬臂浇筑施工（大跨连续梁/连续刚构）
      - 挂篮设计与验算
      - 悬浇节段划分
      - 合龙段施工

6. 关键技术措施
   6.1 大体积混凝土温控措施
   6.2 预应力施工质量控制
   6.3 支架预压与变形监测
   6.4 水中施工措施（围堰/钢平台）

7. 质量控制要点
   - 桩基：成孔质量（孔径/垂直度/沉渣厚度）
   - 混凝土：强度/耐久性/外观
   - 预应力：张拉力/伸长量/压浆饱满度
   - 线形：高程/轴线偏差

8. 安全专项措施
   - 水上作业安全
   - 高处作业安全
   - 大型机械作业安全
   - 交通安全（导改区防护）
```

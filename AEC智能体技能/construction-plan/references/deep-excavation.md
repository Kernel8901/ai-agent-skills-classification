# 深基坑专项施工方案详解

## 一、深基坑支护选型

### 支护形式选择矩阵

| 基坑深度 | 地质条件 | 周边环境 | 推荐支护形式 |
|---------|---------|---------|-------------|
| ≤5m | 土质较好，无地下水 | 开阔 | 放坡开挖（1:0.5-1:1） |
| ≤5m | 一般土质 | 一般 | 土钉墙 |
| 5-10m | 一般土质 | 一般 | 排桩+锚索/支撑 |
| 5-10m | 软土 | 敏感建筑 | 地下连续墙+内支撑 |
| >10m | 软土 | 敏感建筑 | 地下连续墙+多道内支撑 |
| >10m | 一般土质 | 敏感建筑 | 排桩+多道锚索 |

### 止水帷幕选型

| 地层类型 | 地下水位 | 止水帷幕形式 |
|---------|---------|-------------|
| 砂土/粉土 | 高于坑底 | 深层搅拌桩（双排） |
| 砂砾层 | 高于坑底 | 高压旋喷桩 |
| 一般黏土 | 低于坑底 | 可不设止水帷幕 |
| 复杂地层 | 高于坑底 | 地下连续墙（兼作支护） |

## 二、分层开挖工艺参数

### 开挖原则
1. **分层开挖**：每层厚度≤2m
2. **分段开挖**：每段长度≤30m（软土≤20m）
3. **先撑后挖**：支撑达到设计强度80%后方可开挖下层
4. **限时开挖**：软土基坑每层开挖时间≤24h
5. **对称开挖**：长条形基坑对称开挖，减少水平位移

### 典型开挖顺序

**顺作法（放坡+支护）**：
```
第1层：表层土（0-2m）→ 第一道锚索/支撑施工
第2层：第一道支撑以下至-4m → 第二道锚索/支撑施工
第3层：第二道支撑以下至-6m → 第三道锚索/支撑施工
... 依次类推
最后一层：预留200-300mm人工清底
```

**逆作法（市区深基坑）**：
```
顶板施工 → 地下一层开挖 → 地下一层结构施工
→ 地下二层开挖 → 地下二层结构施工
→ ... → 底板施工
```

### 坑底保护措施

| 措施 | 适用条件 | 具体做法 |
|------|---------|---------|
| 预留保护层 | 所有基坑 | 机械开挖预留200-300mm，人工清底 |
| 垫层及时封闭 | 软土/地下水丰富 | 挖至设计标高后24h内浇筑垫层 |
| 坑底加固 | 软土/有承压水 | 水泥搅拌桩/高压旋喷桩封底 |
| 降水减压 | 有承压水 | 深井降水至安全水位 |

## 三、支护计算书框架

### 3.1 计算参数取值

#### 土层参数（来自地勘报告）
```
| 土层名称 | 厚度(m) | γ(kN/m³) | c(kPa) | φ(°) | m值(MN/m⁴) |
|---------|--------|----------|--------|------|-----------|
| 素填土 | 2.0 | 18.0 | 10 | 15 | 2.0 |
| 粉质黏土 | 4.0 | 19.0 | 25 | 18 | 4.5 |
| 淤泥质土 | 6.0 | 17.0 | 12 | 8 | 1.5 |
| ... | ... | ... | ... | ... | ... |
```

#### 荷载取值
- 地面超载：q = 20 kPa（一般情况）/ 30 kPa（有车辆通行）
- 坑边堆载：距坑边≥1.5m，堆载≤20 kPa

### 3.2 整体稳定性验算（瑞典条分法）

**计算公式**：
```
Fs = Σ(cili + Wi·cosθi·tanφi) / Σ(Wi·sinθi)

其中：
- ci：第i条土条底部黏聚力
- li：第i条土条底部长度
- Wi：第i条土条自重
- θi：第i条土条底部倾角
- φi：第i条土条底部内摩擦角

安全系数要求：Fs ≥ 1.30（一级基坑）/ 1.25（二级基坑）/ 1.15（三级基坑）
```

### 3.3 抗倾覆稳定性验算

**计算公式**：
```
Kt = M抗倾 / M倾覆

M抗倾 = 被动土压力对桩底的力矩 + 支撑反力对桩底的力矩
M倾覆 = 主动土压力对桩底的力矩

安全系数要求：Kt ≥ 1.20
```

### 3.4 抗隆起稳定性验算

**Prandtl公式**：
```
Ks = (γD·Nq + c·Nc) / (γ(H+D) + q)

其中：
- D：支护结构入土深度
- H：基坑开挖深度
- Nq、Nc：承载力系数，Nq = tan²(45+φ/2)·e^(πtanφ)
                                 Nc = (Nq-1)/tanφ
- γ：土体重度
- c：土体黏聚力

安全系数要求：Ks ≥ 1.80（一级基坑）/ 1.60（二级基坑）/ 1.40（三级基坑）
```

### 3.5 抗渗流稳定性验算

**计算公式**：
```
Kw = γ'·D / γw·Δh

其中：
- γ'：土体有效重度
- D：支护结构入土深度
- γw：水重度（10 kN/m³）
- Δh：水头差

安全系数要求：Kw ≥ 1.50
```

### 3.6 支护结构内力计算

**方法**：弹性地基梁法（m法）

**基本假定**：
- 桩身视为弹性地基梁
- 地基反力 p = m·z·y（m为地基抗力比例系数，z为深度，y为位移）

**计算步骤**：
1. 划分单元（每单元长度≤1m）
2. 建立刚度矩阵
3. 施加荷载（土压力+水压力+地面超载）
4. 求解位移、内力
5. 绘制弯矩图、剪力图、位移图

**配筋计算**：
```
As = M / (fy·γs·h0)

其中：
- M：计算截面弯矩设计值
- fy：钢筋抗拉强度设计值
- γs：内力臂系数
- h0：截面有效高度
```

### 3.7 锚索承载力计算

**锚索轴向拉力**：
```
Nk = H·s / cosα

其中：
- H：水平分力
- s：锚索水平间距
- α：锚索倾角（一般15-25°）
```

**锚固体承载力**：
```
Nu = π·D·L·τ

其中：
- D：锚固体直径
- L：锚固段长度
- τ：土层与锚固体粘结强度（经验值：黏土50-80kPa，砂土80-150kPa）

安全系数要求：Nk / Nu ≤ 0.5（临时锚索）/ 0.4（永久锚索）
```

## 四、监测预警体系

### 4.1 监测项目设置

| 监测项目 | 一级基坑 | 二级基坑 | 三级基坑 |
|---------|---------|---------|---------|
| 支护结构水平位移 | 必测 | 必测 | 必测 |
| 支护结构沉降 | 必测 | 必测 | 选测 |
| 地下水位 | 必测 | 必测 | 必测 |
| 周边地表沉降 | 必测 | 必测 | 选测 |
| 周边建筑沉降 | 必测 | 必测 | 选测 |
| 立柱沉降 | 必测 | 选测 | 可不测 |
| 支撑轴力 | 必测 | 选测 | 可不测 |
| 锚索拉力 | 必测 | 选测 | 可不测 |
| 土压力 | 选测 | 可不测 | 可不测 |
| 孔隙水压力 | 选测 | 可不测 | 可不测 |

### 4.2 预警值设定

**累计变形预警值**：
| 监测项目 | 一级基坑 | 二级基坑 | 三级基坑 |
|---------|---------|---------|---------|
| 支护结构水平位移 | 30mm | 40mm | 50mm |
| 支护结构沉降 | 20mm | 30mm | 40mm |
| 地表沉降 | 30mm | 40mm | 50mm |
| 建筑沉降 | 20mm | 25mm | 30mm |
| 地下水位变化 | ±500mm | ±500mm | ±500mm |

**变化速率预警值**：
| 监测项目 | 预警值（mm/d） | 报警值（mm/d） |
|---------|--------------|--------------|
| 水平位移 | 2 | 3 |
| 沉降 | 2 | 3 |
| 地下水位 | 500 | 800 |

### 4.3 监测频率

| 施工阶段 | 监测频率 |
|---------|---------|
| 基坑开挖前 | 至少2次初始值 |
| 开挖深度≤5m | 1次/2天 |
| 开挖深度>5m | 1次/天 |
| 底板浇筑后7天内 | 1次/天 |
| 底板浇筑后7-14天 | 1次/2天 |
| 底板浇筑后14-28天 | 1次/3天 |
| 底板浇筑后>28天 | 1次/周 |
| 出现异常 | 加密至2-4次/天 |

### 4.4 信息化施工响应机制

**黄色预警（累计值达预警值70%）**：
- 加密监测频率
- 分析原因，调整施工参数
- 检查监测数据可靠性

**橙色预警（累计值达预警值）**：
- 暂停开挖
- 召开技术分析会
- 采取加固措施
- 持续监测至稳定

**红色报警（累计值达报警值或速率突增）**：
- 立即停止所有施工
- 撤离相关人员
- 启动应急预案
- 必要时回填

## 五、应急预案

### 5.1 坑壁坍塌应急处置

**预警征兆**：
- 支护结构水平位移速率突增
- 坑壁出现裂缝并扩展
- 支护结构出现异响

**应急措施**：
1. 立即停止坑内作业，撤离人员
2. 疏散坑边临时设施内人员
3. 在坍塌区域外围设置警戒线
4. 回填反压（砂袋/土方）
5. 加密监测，评估扩大范围
6. 查明原因，制定加固方案

### 5.2 突涌水应急处置

**预警征兆**：
- 坑底出现冒水/冒砂
- 地下水位异常下降
- 坑边地面开裂塌陷

**应急措施**：
1. 立即停止坑内作业
2. 启用备用降水井，加大降水力度
3. 坑内设集水坑抽排
4. 封堵涌水点（必要时注入水玻璃）
5. 坑底压重（砂袋/混凝土）
6. 分析原因，必要时增设止水帷幕

### 5.3 周边建筑沉降超限应急处置

**预警征兆**：
- 建筑沉降达预警值
- 建筑墙体出现裂缝
- 管线接口渗漏

**应急措施**：
1. 加密建筑监测点
2. 建筑基础注浆加固
3. 调整基坑施工方案（减小开挖段长、及时支护）
4. 必要时对建筑进行临时支撑
5. 协调居民临时转移

## 六、验收要点

### 6.1 分项验收

| 验收项目 | 验收内容 | 验收标准 |
|---------|---------|---------|
| 桩基施工 | 桩位偏差、垂直度、桩长 | 桩位偏差≤50mm，垂直度≤1% |
| 冠梁施工 | 截面尺寸、钢筋、混凝土强度 | 截面偏差±20mm，强度≥设计值 |
| 锚索施工 | 孔位、孔径、孔深、注浆 | 孔位偏差≤100mm，注浆饱满 |
| 土钉施工 | 孔位、孔径、孔深、注浆 | 孔位偏差≤100mm，注浆饱满 |
| 土方开挖 | 标高、平面尺寸、边坡坡度 | 标高偏差0~-50mm |
| 降水运行 | 水位降深、含砂率 | 水位低于设计要求，含砂率<1/50000 |

### 6.2 监测验收

- 监测点布置符合设计要求
- 初始值采集准确（至少2次）
- 监测数据连续、完整
- 预警机制响应及时

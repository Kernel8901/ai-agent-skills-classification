# Calculation Guide

## 常用公式

### 1. 基坑稳定性计算

#### 1.1 抗倾覆稳定性
**公式**: K = Mr / Mo

其中：
- K: 抗倾覆安全系数 (≥1.2)
- Mr: 抗倾覆力矩 (kN·m)
- Mo: 倾覆力矩 (kN·m)

**计算步骤**：
1. 计算主动土压力合力Ea
2. 计算被动土压力合力Ep
3. 计算力矩，验算K值

**示例**：
```
基坑深度 h=10m
土重度 γ=18kN/m³
内摩擦角 φ=25°
计算得 Ea=225kN/m
Ep=320kN/m
K = (320×3)/(225×5) = 0.85 < 1.2 → 不满足
→ 需增加支护措施
```

#### 1.2 整体稳定性
**公式**: K = ΣMs / ΣMt

其中：
- K: 整体稳定安全系数 (≥1.25)
- Ms: 抗滑力矩
- Mt: 滑动力矩

### 2. 支架计算

#### 2.1 门式支架承载力
**公式**: N = A × f

其中：
- N: 立杆承载力 (kN)
- A: 立杆截面积 (mm²)
- f: 钢材设计强度 (N/mm²)

**示例**：
```
φ48×3.5mm钢管
截面积 A=489mm²
Q235钢 f=205N/mm²
N = 489×205 = 100.3kN
→ 单杆承载力约100kN
```

#### 2.2 模板强度验算
**公式**: σ = M / W ≤ f

其中：
- σ: 弯曲应力 (N/mm²)
- M: 最大弯矩 (N·mm)
- W: 截面抵抗矩 (mm³)
- f: 模板材设计强度

**验算步骤**：
1. 计算荷载组合
2. 确定最不利截面
3. 计算弯矩和剪力
4. 验算强度和刚度

---

### 3. 钢筋工程

#### 3.1 锚固长度
**公式**: La = α × d × fy / ft

其中：
- La: 锚固长度 (mm)
- α: 钢筋外形系数 (光圆=0.16, 带肋=0.14)
- d: 钢筋直径 (mm)
- fy: 钢筋设计强度 (N/mm²)
- ft: 混凝土轴心抗拉强度 (N/mm²)

**示例**：
```
HRB400 φ20
ft = 1.71N/mm² (C30)
La = 0.14×20×360/1.7 = 592mm
→ 取40d=800mm
```

## 计算流程

### 深基坑支护方案计算流程

1. **收集资料**
   - 地质勘察报告
   - 地下室结构图纸
   - 周边环境资料

2. **确定参数**
   - 土层物理力学参数
   - 地面荷载取值
   - 水位设计条件

3. **计算分析**
   - 土压力计算
   - 内力分析
   - 稳定性验算

4. **方案设计**
   - 支护结构选型
   - 构件设计计算
   - 构造措施确定

5. **成果整理**
   - 计算书编制
   - 施工图绘制
   - 方案文本编写

### 高支模计算流程

1. **荷载统计**
   - 恒荷载：模板+钢筋+混凝土
   - 活荷载：施工人员+机具+振捣

2. **支架设计**
   - 立杆布置间距
   - 水平杆步距
   - 剪刀撑设置

3. **验算项目**
   - 立杆稳定性
   - 梁底模强度
   - 板底模刚度
   - 整体稳定性

## 注意事项

### 计算要点
1. 荷载取值要完整，考虑施工全过程
2. 计算模型要符合实际传力路径
3. 组合荷载时要考虑最不利情况
4. 计算结果要与规范限值对比

### 常见问题
1. 荷载漏项：漏算施工荷载、振捣荷载
2. 组合不当：未考虑多种荷载同时作用
3. 模型简化不当：计算模型与实际不符
4. 结果判定错误：未正确引用规范限值

### 计算软件推荐
- 理正深基坑：基坑支护计算
- MIDAS Gen：结构整体分析
- PKPM：模板支架设计
- SAP2000：复杂结构分析

### 计算资料存档
- 计算书需打印签字
- 电子版与纸质版一致
- 保留计算依据（规范、图集）
- 变更计算需保留修改记录

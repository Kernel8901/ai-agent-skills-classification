# 桥梁工程施工方案详解

## 一、桥梁工程分类与特点

### 1.1 按结构体系分类

| 桥型 | 适用跨径 | 结构特点 | 施工方法 |
|------|---------|---------|---------|
| 梁桥 | 20-200m | 受弯为主 | 预制安装/现浇/顶推 |
| 拱桥 | 50-500m | 受压为主 | 缆索吊装/转体施工 |
| 斜拉桥 | 200-1000m | 塔索梁组合 | 悬臂拼装/悬臂浇筑 |
| 悬索桥 | 500-2000m | 悬索受拉 | 猫道法架设 |
| 刚构桥 | 50-300m | 墩梁固结 | 悬臂浇筑 |

### 1.2 按材料分类

| 类型 | 特点 | 适用范围 |
|------|------|---------|
| 钢筋混凝土桥 | 造价低，自重大 | 小跨径桥梁 |
| 预应力混凝土桥 | 跨越能力大，应用广 | 中大跨径桥梁 |
| 钢桥 | 自重轻，施工快 | 大跨径、临时桥 |
| 组合桥 | 发挥材料优势 | 各类桥梁 |

## 二、下部结构施工

### 2.1 桩基础施工

#### 钻孔灌注桩

**工艺流程**：
```
测量定位 → 埋设护筒 → 钻机就位 → 钻进成孔
→ 清孔 → 钢筋笼下放 → 二次清孔 → 灌注混凝土
```

**质量控制要点**：

| 控制项目 | 允许偏差 | 检测方法 |
|---------|---------|---------|
| 桩位偏差 | ≤50mm | 测量 |
| 孔径 | 不小于设计 | 检孔器 |
| 垂直度 | ≤1% | 测斜仪 |
| 孔深 | 不小于设计 | 测绳 |
| 沉渣厚度 | ≤100mm（摩擦桩）/ ≤50mm（端承桩） | 测绳 |

**常见问题处理**：

| 问题 | 原因 | 处理措施 |
|------|------|---------|
| 坍孔 | 地层松散、泥浆比重不足 | 提高泥浆比重、下套管护壁 |
| 斜孔 | 钻机不稳、地层软硬不均 | 调整钻机、扫孔纠偏 |
| 沉渣过厚 | 清孔不彻底 | 二次清孔 |
| 导管堵塞 | 混凝土和易性差 | 提升导管、疏通 |

#### 预应力管桩

**施工方法**：
- 锤击法：适用于一般土层
- 静压法：适用于对振动敏感区域

**质量控制**：
| 项目 | 要求 |
|------|------|
| 桩位偏差 | ≤100mm |
| 垂直度 | ≤0.5% |
| 入土深度 | 符合设计 |
| 最后贯入度 | 符合设计或试桩确定 |

### 2.2 承台施工

**基坑开挖与支护**：
- 陆上承台：放坡开挖/钢板桩支护
- 水中承台：围堰（钢板桩/钢套箱）

**承台施工流程**：
```
基坑开挖 → 基础处理 → 垫层浇筑 → 桩头处理
→ 钢筋绑扎 → 模板安装 → 混凝土浇筑 → 养护
```

**大体积混凝土温控措施**：

| 措施 | 具体做法 |
|------|---------|
| 优化配合比 | 降低水泥用量，掺粉煤灰、矿粉 |
| 控制入模温度 | ≤28℃（夏季加冰搅拌） |
| 分层浇筑 | 每层≤300mm |
| 埋设冷却水管 | 通水降温，进出水温差≤25℃ |
| 保温养护 | 覆盖保温材料，内外温差≤25℃ |

### 2.3 墩柱施工

#### 模板体系选择

| 模板类型 | 适用高度 | 特点 |
|---------|---------|------|
| 组合钢模 | ≤10m | 造价低，拼装费时 |
| 大钢模 | 10-30m | 整体性好，质量高 |
| 翻模 | 30-50m | 分段施工，安全风险较高 |
| 滑模 | >50m | 连续施工，质量要求高 |
| 爬模 | >50m | 安全性好，应用广泛 |

#### 墩柱施工流程

```
测量放样 → 钢筋绑扎 → 模板安装 → 混凝土浇筑
→ 养护 → 模板提升（翻模/爬模）→ 下一节段施工
```

**混凝土浇筑要求**：
- 分层浇筑，每层≤300mm
- 对称下料，防止偏压
- 连续浇筑，间隔时间≤2h
- 坍落度控制在160-200mm

### 2.4 盖梁施工

#### 支架形式

| 支架类型 | 适用条件 | 特点 |
|---------|---------|------|
| 落地支架 | 低墩、地面条件好 | 简单可靠 |
| 抱箍法 | 高墩、地面条件差 | 不占地面空间 |
| 托架法 | 墩顶悬出部分 | 需预埋件 |

#### 预应力张拉

**张拉顺序**：先长束后短束，先下后上，对称张拉

**张拉控制**：
```
张拉力 = 设计张拉力 × 锚口损失系数（约1.03-1.05）

伸长量校核：
实际伸长量 = (10%σ→100%σ测量伸长值) + 初应力推算伸长值
允许偏差：±6%
```

## 三、上部结构施工

### 3.1 预制梁施工

#### 预制场布置

**选址原则**：
- 靠近桥位，减少运距
- 地基承载力满足要求
- 便于材料进场

**台座设置**：
| 项目 | 要求 |
|------|------|
| 数量 | 根据工期、台座周转周期计算 |
| 间距 | ≥2倍梁高 |
| 基础 | 承载力≥200kPa，设置排水沟 |
| 预拱度 | 按设计设置 |

#### 预制梁施工流程

```
底模安装 → 钢筋绑扎（底腹板）→ 波纹管安装
→ 内模安装 → 钢筋绑扎（顶板）→ 侧模安装
→ 混凝土浇筑 → 养护 → 拆模 → 预应力张拉
→ 压浆封锚 → 移梁存放
```

#### 预应力施工要点

**张拉时机**：
- 混凝土强度≥设计强度90%
- 弹性模量≥设计值90%

**张拉程序**：
```
0 → 初应力(10%σ) → 量伸长值 → 张拉至控制应力
→ 持荷5min → 量伸长值 → 锚固
```

**孔道压浆**：
- 水泥浆水灰比：0.26-0.28
- 掺入高效减水剂和膨胀剂
- 压浆压力：0.5-0.7MPa
- 压浆后48h内温度≥5℃

#### 存梁与运输

**存梁要求**：
| 项目 | 要求 |
|------|------|
| 支点位置 | 距梁端0.5-1.0m |
| 堆放层数 | ≤3层 |
| 垫块 | 同一垂直面 |

**运输方式**：
- 轮胎式运梁车：适用于较长距离
- 履带式运梁车：适用于软基道路
- 轨道式运梁台车：适用于标准化预制场

#### 架梁方法

| 方法 | 适用条件 | 特点 |
|------|---------|------|
| 吊车架设 | 跨径≤30m，岸上作业 | 灵活、造价低 |
| 架桥机架设 | 标准跨径，长桥 | 效率高、标准化 |
| 龙门吊架设 | 预制场内或栈桥上 | 安全、可控 |

**架设顺序**：先边跨后中跨，对称进行

### 3.2 现浇箱梁施工

#### 支架方案

**满堂支架**：
- 适用：桥下无通行要求
- 材料：碗扣式/盘扣式钢管架
- 地基处理：硬化+垫层

**梁柱式支架**：
- 适用：桥下有通行要求
- 材料：贝雷片/钢桁梁+钢管柱
- 基础：桩基/扩大基础

#### 支架预压

| 项目 | 要求 |
|------|------|
| 预压荷载 | 1.1倍梁体自重 |
| 加载分级 | 60%→80%→100%→110% |
| 稳定标准 | 24h沉降≤1mm |
| 卸载 | 分级卸载 |

#### 箱梁浇筑

**浇筑方案**：
| 方案 | 适用条件 | 特点 |
|------|---------|------|
| 一次浇筑 | 跨径较小、工期紧 | 需要大量人力 |
| 分两次浇筑 | 常规方案 | 先底腹板，后顶板 |
| 分段浇筑 | 大跨径 | 需设施工缝、齿块 |

**浇筑顺序**：
```
纵向：从低处向高处（顺桥向坡度）
横向：先底板，后腹板，最后顶板
分层：每层≤300mm
```

### 3.3 悬臂浇筑施工

#### 适用范围
- 大跨径预应力混凝土连续梁、连续刚构
- 跨径通常100-300m

#### 挂篮设计

**挂篮类型**：
| 类型 | 承载力 | 特点 |
|------|--------|------|
| 菱形挂篮 | 100-400t | 刚度大、变形小 |
| 三角形挂篮 | 80-300t | 重量轻、拼装快 |
| 滑动斜拉式 | 50-200t | 结构简单 |

**挂篮验算内容**：
- 承重能力
- 抗倾覆稳定性（安全系数≥2.0）
- 锚固系统承载力
- 走行系统

#### 悬浇施工流程

```
0#块施工（墩顶）→ 挂篮拼装 → 1#块浇筑
→ 预应力张拉 → 挂篮前移 → 下一块浇筑
→ ... → 边跨合龙 → 中跨合龙 → 体系转换
```

#### 悬浇控制要点

| 项目 | 控制目标 | 措施 |
|------|---------|------|
| 线形控制 | 标高偏差≤15mm | 施工监测、参数调整 |
| 应力控制 | 不超限 | 埋设应变计、实时监测 |
| 合龙精度 | 轴线偏差≤10mm，高差≤15mm | 温度选择、配重调整 |

#### 合龙段施工

**合龙顺序**：先边跨后中跨

**合龙要点**：
| 项目 | 要求 |
|------|------|
| 合龙时间 | 选择一天中气温最低、稳定的时段 |
| 临时锁定 | 焊接劲性骨架或张拉临时束 |
| 配重 | 浇筑混凝土同时等量卸载 |
| 混凝土 | 微膨胀、早强混凝土 |

## 四、预应力施工质量控制

### 4.1 原材料检验

| 材料 | 检验项目 | 频率 |
|------|---------|------|
| 钢绞线 | 抗拉强度、伸长率、弹性模量 | 每批≤60t |
| 锚具 | 硬度、静载锚固性能 | 每批≤1000套 |
| 波纹管 | 外观、径向刚度、抗渗漏 | 每批≤50000m |

### 4.2 张拉质量控制

**双控指标**：
- 张拉力：油表读数控制
- 伸长量：实测伸长量与理论伸长量偏差≤±6%

**伸长量计算**：
```
ΔL = (P·L) / (Ap·Ep)

其中：
- P：平均张拉力（考虑孔道摩阻损失）
- L：孔道长度
- Ap：预应力筋截面积
- Ep：预应力筋弹性模量
```

### 4.3 压浆质量控制

| 检测项目 | 要求 |
|---------|------|
| 水泥浆流动度 | 12-18s |
| 水泥浆泌水率 | ≤1% |
| 水泥浆膨胀率 | 0-3% |
| 试块强度 | ≥50MPa（28d） |

## 五、桥梁检测与验收

### 5.1 成桥检测

| 检测项目 | 检测内容 |
|---------|---------|
| 静载试验 | 挠度、应变、裂缝 |
| 动载试验 | 自振频率、阻尼比、冲击系数 |
| 外观检查 | 混凝土外观、几何尺寸、线形 |

### 5.2 质量验收标准

| 项目 | 允许偏差 |
|------|---------|
| 轴线偏位 | ≤10mm |
| 桥面标高 | ±10mm |
| 桥面宽度 | ±20mm |
| 桥面平整度 | ≤3mm（3m直尺） |
| 支座位置 | ≤5mm |

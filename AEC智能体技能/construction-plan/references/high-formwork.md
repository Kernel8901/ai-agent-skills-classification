# 高支模专项施工方案详解

## 一、高支模判定标准

### 危大工程判定（需编制专项方案）

满足以下任一条件：

| 条件 | 数值 |
|------|------|
| 搭设高度 | ≥8m |
| 搭设跨度 | ≥18m |
| 施工总荷载（设计值） | ≥15kN/m² |
| 集中线荷载（设计值） | ≥20kN/m |

### 超危大工程判定（需专家论证）

满足以下任一条件：

| 条件 | 数值 |
|------|------|
| 搭设高度 | ≥18m |
| 搭设跨度 | ≥36m |
| 施工总荷载（设计值） | ≥20kN/m² |
| 集中线荷载（设计值） | ≥30kN/m |

## 二、支撑体系设计参数

### 2.1 立杆参数

| 参数 | 一般要求 | 高支模要求 |
|------|---------|-----------|
| 立杆间距 | ≤1.5m | ≤1.2m（梁下） / ≤1.5m（板下） |
| 步距 | ≤1.8m | ≤1.5m |
| 扫地杆距地面 | ≤200mm | ≤200mm |
| 顶托伸出长度 | ≤300mm | ≤200mm（高支模） |
| 顶托插入长度 | ≥150mm | ≥200mm |

### 2.2 剪刀撑设置

**竖向剪刀撑**（高支模必须设置）：

| 设置位置 | 要求 |
|---------|------|
| 四周 | 满布 |
| 内部 | 每隔6-8m设置一道 |
| 与地面夹角 | 45°-60° |

**水平剪刀撑**（高支模必须设置）：

| 设置位置 | 要求 |
|---------|------|
| 扫地杆位置 | 设置一道 |
| 中间位置 | 每隔6-8m设置一道 |
| 顶板下方 | 设置一道 |

### 2.3 扣件式钢管架材料要求

| 材料 | 规格 | 允许偏差 |
|------|------|---------|
| 钢管 | φ48.3×3.6mm | 壁厚≥3.0mm |
| 扣件 | 可锻铸铁/钢板冲压 | 重量≥1.0kg/个 |
| 顶托 | 实心/空心 | 插入立杆长度≥150mm |

## 三、荷载计算

### 3.1 荷载分类与取值

**永久荷载（G）**：

| 荷载项目 | 计算公式 | 备注 |
|---------|---------|------|
| 模板自重 | G₁ = 0.3-0.5 kN/m² | 按实际采用 |
| 钢筋自重 | G₂ = 梁每m³混凝土1.5kN / 板每m³混凝土1.1kN | |
| 新浇混凝土自重 | G₃ = 24-25 kN/m³ | 普通混凝土取24 |
| 支架自重 | G₄ = 按实际计算 | 可估算为0.1-0.2kN/m²/每米高度 |

**可变荷载（Q）**：

| 荷载项目 | 取值 | 备注 |
|---------|------|------|
| 施工人员及设备荷载 | Q₁ = 2.5 kN/m² | 计算支架立杆时 |
| 施工人员及设备荷载 | Q₁ = 1.0 kN/m² | 计算模板及小楞时 |
| 振捣混凝土荷载 | Q₂ = 2.0 kN/m² | 水平模板 |
| 倾倒混凝土荷载 | Q₃ = 2.0-4.0 kN/m² | 视浇筑方式而定 |

### 3.2 荷载组合

**承载力计算（基本组合）**：

```
S = 1.35ΣγG·Gk + 1.4γQ·ΣQk

简化计算：
S = 1.35G + 1.4Q

其中：
- G：永久荷载标准值
- Q：可变荷载标准值
```

**变形验算（标准组合）**：

```
S = ΣGk + ΣQk
```

### 3.3 荷载计算示例

**梁模板支架荷载计算**：

```
梁截面：300mm×800mm
梁跨度：6m
支撑高度：10m

永久荷载标准值：
- 模板自重：0.5 kN/m² × 0.3m = 0.15 kN/m
- 钢筋自重：1.5 kN/m³ × 0.3×0.8 = 0.36 kN/m
- 混凝土自重：24 kN/m³ × 0.3×0.8 = 5.76 kN/m
- 支架自重：0.15 kN/m²/m × 10m × 0.3m = 0.45 kN/m
  合计 Gk = 6.72 kN/m

可变荷载标准值：
- 施工荷载：2.5 kN/m² × 0.3m = 0.75 kN/m
- 振捣荷载：2.0 kN/m² × 0.3m = 0.60 kN/m
  合计 Qk = 1.35 kN/m

荷载设计值：
S = 1.35 × 6.72 + 1.4 × 1.35 = 9.07 + 1.89 = 10.96 kN/m
```

**板模板支架荷载计算**：

```
板厚：150mm
支撑高度：10m
立杆间距：1.0m×1.0m

永久荷载标准值：
- 模板自重：0.5 kN/m²
- 钢筋自重：1.1 kN/m³ × 0.15 = 0.165 kN/m²
- 混凝土自重：24 kN/m³ × 0.15 = 3.6 kN/m²
- 支架自重：0.15 kN/m²/m × 10m = 1.5 kN/m²
  合计 Gk = 5.765 kN/m²

可变荷载标准值：
- 施工荷载：2.5 kN/m²
- 振捣荷载：2.0 kN/m²
  合计 Qk = 4.5 kN/m²

荷载设计值：
S = 1.35 × 5.765 + 1.4 × 4.5 = 7.78 + 6.3 = 14.08 kN/m²

单根立杆承受荷载：
N = 14.08 × 1.0 × 1.0 = 14.08 kN
```

## 四、立杆稳定性验算

### 4.1 计算长度确定

**不考虑风荷载时**：

```
l0 = k × μ1 × μ2 × h

其中：
- k：计算长度附加系数，取1.155
- μ1：考虑步距影响的计算长度系数，查表（约1.5-2.0）
- μ2：考虑立杆伸出顶层水平杆长度的系数
  μ2 = 1 + 2a/h（a为伸出长度，h为步距）
- h：步距

简化计算：
l0 = k × μ × h = 1.155 × 1.5 × 1.5 = 2.60m
```

**考虑风荷载时**：

```
l0 = max(k·μ·h, k·μ'·h)

μ'：考虑风荷载作用的计算长度系数
```

### 4.2 长细比计算

```
λ = l0 / i

其中：
- i：回转半径，φ48.3钢管 i = 15.9mm
- λ ≤ 230（容许长细比）

示例：
λ = 2600 / 15.9 = 163.5 < 230，满足要求
```

### 4.3 稳定系数φ确定

根据长细比λ查表（《钢结构设计标准》GB 50017）：

| λ | φ（Q235钢） |
|---|------------|
| 140 | 0.387 |
| 150 | 0.339 |
| 160 | 0.298 |
| 170 | 0.263 |
| 180 | 0.233 |

**插值计算**：λ = 163.5 → φ ≈ 0.283

### 4.4 稳定性验算公式

```
N / (φ·A) ≤ f

其中：
- N：轴向压力设计值（kN）
- φ：轴心受压稳定系数
- A：立杆截面积，φ48.3×3.6钢管 A = 506mm²
- f：钢材抗压强度设计值，Q235钢 f = 205 N/mm²

示例验算：
N / (φ·A) = 14080 / (0.283 × 506) = 98.2 N/mm² < 205 N/mm²

结论：稳定性满足要求
```

### 4.5 安全系数校核

```
K = φ·A·f / N

安全系数要求：K ≥ 2.0（临时结构）

示例：
K = 0.283 × 506 × 205 / 14.08 = 2.09 > 2.0

结论：安全系数满足要求
```

## 五、扣件抗滑验算

### 5.1 直角扣件抗滑

```
R ≤ Rc

其中：
- R：扣件传递的力设计值
- Rc：扣件抗滑承载力设计值，取8kN

示例：
纵向水平杆传给立杆的力 R = N/2 = 7.04 kN < 8 kN

结论：满足要求
```

### 5.2 旋转扣件抗滑

```
当直角扣件不满足时，可采用双扣件：
R ≤ 2Rc = 16kN
```

## 六、地基承载力验算

### 6.1 基础底面积确定

```
A0 = N / (fg - γ·d)

其中：
- A0：所需基础底面积
- N：立杆传至基础的力
- fg：地基承载力特征值
- γ：土的重度
- d：基础埋深（可忽略）

立杆底座尺寸：100mm×100mm = 10000mm²
垫板尺寸：≥100mm×100mm×50mm
```

### 6.2 地基承载力要求

| 地基类型 | 承载力特征值 fg |
|---------|----------------|
| 素填土（压实） | 80-120 kPa |
| 黏土（硬塑） | 150-200 kPa |
| 砂土（中密） | 150-250 kPa |
| 混凝土垫层 | ≥150 kPa |

### 6.3 验算示例

```
N = 14.08 kN
垫板尺寸：200mm×200mm

基础底面平均压力：
p = N / A = 14.08 × 10³ / (200×200) = 0.352 N/mm² = 352 kPa

需设置C20混凝土垫层或垫板加厚
```

## 七、支架预压

### 7.1 预压目的

1. 检验支架承载力安全性
2. 消除支架非弹性变形
3. 收集弹性变形数据，设置预拱度

### 7.2 预压荷载

| 预压阶段 | 荷载比例 |
|---------|---------|
| 第一级 | 60%设计荷载 |
| 第二级 | 80%设计荷载 |
| 第三级 | 100%设计荷载 |

### 7.3 预压观测

| 观测项目 | 观测频率 | 稳定标准 |
|---------|---------|---------|
| 支架沉降 | 每级加载前后、稳定后 | 24h沉降<1mm |
| 支架位移 | 同上 | 无明显位移 |

### 7.4 卸载要求

- 沉降稳定后方可卸载
- 分级卸载（100%→60%→0）
- 卸载后测量弹性变形量

## 八、混凝土浇筑控制

### 8.1 浇筑顺序

**梁板结构**：
```
先浇筑柱→再浇筑梁（从跨中向两端）→最后浇筑板
```

**大跨度梁**：
```
分层浇筑，每层≤300mm
对称浇筑，避免偏载
```

### 8.2 浇筑速度控制

| 混凝土类型 | 最大浇筑速度 |
|-----------|-------------|
| 普通混凝土 | 2.0 m/h |
| 大流动性混凝土 | 1.5 m/h |

### 8.3 浇筑过程监测

- 专人监测支架变形
- 发现异常立即停止浇筑
- 监测点布置：跨中、1/4跨、支座处

---

## 九、拆除要求

### 9.1 拆除条件

**混凝土强度要求**：

| 构件类型 | 跨度 | 拆模时混凝土强度 |
|---------|------|----------------|
| 板 | ≤2m | ≥50%设计强度 |
| 板 | >2m, ≤8m | ≥75%设计强度 |
| 板 | >8m | ≥100%设计强度 |
| 梁、拱、壳 | ≤8m | ≥75%设计强度 |
| 梁、拱、壳 | >8m | ≥100%设计强度 |
| 悬臂构件 | - | ≥100%设计强度 |

### 9.2 拆除顺序

```
1. 先拆除非承重模板（侧面模板）
2. 后拆除承重模板（底模）
3. 从跨中向支座方向拆除
4. 拆除的模板、支架及时清运
```

### 9.3 安全要求

- 设置警戒区，禁止人员进入
- 专人指挥，统一行动
- 逐层拆除，禁止大面积整体推倒
- 拆除人员系安全带

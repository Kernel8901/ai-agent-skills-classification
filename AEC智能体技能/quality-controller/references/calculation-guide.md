# Calculation Guide

## 质量指标计算

### 1. 质量合格率

#### 1.1 分项工程合格率
**公式**: 合格率 = (合格数量 / 检查总数) × 100%

**示例**：
```
检查钢筋绑扎100个点
合格点位数95个
合格率 = 95/100 × 100% = 95%
```

#### 1.2 混凝土强度评定
**公式**: fcu,k ≥ fcu,n + λ1 × S

其中：
- fcu,k：混凝土强度标准值
- fcu,n：n组试件强度平均值
- λ1：合格判定系数（查规范）
- S：n组试件强度标准差

---

### 2. 质量控制指标

#### 2.1 质量成本
**公式**: 质量成本 = 预防成本 + 鉴定成本 + 内部故障成本 + 外部故障成本

**最优质量成本**：预防成本和鉴定成本与故障成本之和最低

#### 2.2 质量损失率
**公式**: 质量损失率 = 质量损失金额 / 产值 × 100%

---

### 3. 抽样检验

#### 3.1 抽样数量计算
**公式**: n = (t × σ / E)²

其中：
- n：抽样数量
- t：置信度系数
- σ：标准差
- E：允许误差

#### 3.2 合格判定
**公式**: n0 - c ≤ d ≤ n0 + c

其中：
- n0：抽样数量
- c：允许不合格数
- d：不合格数

---

### 4. 质量指标统计

#### 4.1 常用质量指标

| 指标名称 | 计算方法 | 目标值 |
|---------|---------|-------|
| 分项工程合格率 | 合格数/检查总数 | ≥95% |
| 隐蔽工程验收率 | 验收数/应验收数 | 100% |
| 质量问题整改率 | 已整改数/问题总数 | ≥98% |
| 质量投诉率 | 投诉次数/交付户数 | ≤1% |

#### 4.2 质量评分
**公式**: 质量得分 = Σ(分项得分 × 权重) / Σ权重

## 质量检测方法

### 1. 回弹法检测混凝土强度
- 适用范围：自然养护条件下混凝土
- 检测数量：不少于10个测区
- 数据处理：剔除最大值和最小值，取平均值

### 2. 超声法检测混凝土缺陷
- 检测内容：裂缝深度、内部缺陷
- 检测原理：超声波传播速度和时间
- 判定标准：根据波速和波形判断

### 3. 钢筋保护层厚度检测
- 检测方法：电磁感应法
- 合格率判定：90%以上测点合格

## 注意事项

### 计算要点
1. 统计数据要真实准确
2. 样本要有代表性
3. 判定要符合规范要求

### 常见问题
1. 抽样数量不足
2. 数据处理方法错误
3. 判定标准引用不当

### 改进建议
- 规范检测方法
- 完善记录台账
- 定期统计分析

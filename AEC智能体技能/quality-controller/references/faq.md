# FAQ

## 常见问题

### Q1: 质量三检制是什么？

**A**：三检制是指：
- **自检**：操作者自己检查
- **互检**：班组之间互相检查
- **交接检**：上下工序之间的检查

三检制是施工质量控制的基本制度，要严格执行。

### Q2: 隐蔽工程验收有哪些？

**A**：隐蔽工程包括：
1. 地基与基础
2. 主体结构
3. 防水工程
4. 保温工程
5. 电气管线
6. 给排水工程

隐蔽工程验收要在隐蔽前进行，验收合格后方可进行下道工序。

### Q3: 质量问题如何处理？

**A**：质量问题处理流程：
1. **识别**：发现质量问题
2. **标识**：标记问题部位
3. **报告**：上报质量问题
4. **处置**：制定整改方案
5. **整改**：按方案整改
6. **验收**：验证整改效果
7. **归档**：记录处理过程

### Q4: 质量验收程序是什么？

**A**：质量验收程序：
1. **分项工程验收**：工序完成后
2. **分部工程验收**：子分部完成后
3. **单位工程验收**：全部完成后
4. **竣工验收**：建设单位组织

### Q5: 如何预防质量通病？

**A**：预防措施：
1. **严格原材料进场检验**
2. **执行样板先行制度**
3. **加强过程控制**
4. **做好技术交底**
5. **及时处理质量问题**

### Q6: 质量资料包括哪些？

**A**：质量资料：
1. 施工日志
2. 质量检验批记录
3. 隐蔽工程验收记录
4. 原材料进场检验记录
5. 试验报告
6. 质量问题处理记录

## 疑难解答

| 问题 | 解决方案 |
|------|----------|
| 混凝土强度不足 | 回弹检测，制定加固方案 |
| 钢筋保护层不足 | 凿除重新绑扎或加固处理 |
| 防水渗漏 | 注浆堵漏，重新做防水层 |
| 裂缝 | 分析原因，采用灌浆或封闭处理 |

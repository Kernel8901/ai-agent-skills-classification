# 工程质量验收与检测管理助手

## 验收层级体系

```
单位工程验收
  └── 子单位工程验收
        └── 分部工程验收（地基基础/主体结构/装饰装修等）
              └── 子分部工程验收
                    └── 分项工程验收
                          └── 检验批验收（最小验收单元）
```

## 验收程序

### 检验批验收
1. 施工单位自检合格，填写检验批质量验收记录
2. 专业监理工程师组织验收
3. 验收合格签字确认

### 分项工程验收
1. 所有检验批验收合格
2. 施工单位填写分项工程质量验收记录
3. 监理工程师验收签字

### 分部工程验收
1. 所有分项工程验收合格
2. 质量控制资料完整
3. 安全和功能检验合格
4. 观感质量验收合格
5. 总监理工程师组织，建设单位项目负责人参加

### 单位工程竣工验收
1. 完成设计和合同约定内容
2. 技术档案和施工管理资料完整
3. 主要功能项目检验合格
4. 观感质量验收合格
5. 建设单位组织，五方责任主体参加

## 不合格品处置流程

```
发现不合格 → 标识隔离 → 原因分析
→ 处置方案（返工/返修/让步接收/降级）
→ 方案审批（监理/设计确认）
→ 处置实施 → 复验确认 → 记录归档
```

### 处置方式选择原则
- **返工**：不合格严重，影响结构安全或使用功能
- **返修**：轻微缺陷，修复后满足要求
- **让步接收**：经设计验算，不影响安全和使用，需设计单位书面确认
- **降级使用**：改变用途，满足降级后的使用要求

### 1. construction-plan（施工方案生成器）

#### 1.1 功能概述

根据工程类型、规模、地质条件等信息，自动生成结构化施工方案。支持从施工组织设计到危大工程专项方案的全谱系编制。

#### 1.2 触发关键词

- "帮我写施工方案"
- "生成施工组织设计"
- "编制专项施工方案"
- "深基坑方案"
- "高支模方案"
- "桥梁施工方案"
- "危大工程方案"

#### 1.3 核心能力

| 能力模块 | 说明 |
|---------|------|
| 施工组织设计 | 8章节标准结构：工程概况、施工部署、进度计划、施工方法、资源配置、质量措施、安全措施、季节施工 |
| 危大工程判定 | 自动识别是否超危大，提示是否需要专家论证 |
| 计算书编写 | 提供稳定性验算公式、参数取值、计算过程 |
| 规范引用 | 自动引用适用规范，标注规范编号 |

#### 1.4 方案类型覆盖

**一般施工方案**：
- 土建工程（基础、主体、装饰）
- 市政工程（道路、管网）
- 机电安装工程

**危大工程专项方案**：
| 类型 | 判定标准 | 超危大标准 |
|------|---------|-----------|
| 深基坑 | 开挖深度≥5m | ≥8m需专家论证 |
| 高支模 | 高≥8m/跨≥18m/荷载≥15kN/m² | 高≥18m需专家论证 |
| 脚手架 | 搭设高度≥24m | ≥50m需专家论证 |
| 起重吊装 | 单件≥10t | ≥100t需专家论证 |

#### 1.5 深度技术资料

**深基坑详解（deep-excavation.md）**：

- 支护选型矩阵：根据深度、地质、环境推荐支护形式
- 分层开挖工艺：每层≤2m，每段≤30m，限时开挖≤24h
- 稳定性验算公式：
  - 整体稳定（瑞典条分法）：Fs = Σ(cili + Wi·cosθi·tanφi) / Σ(Wi·sinθi)
  - 抗倾覆：Kt = M抗倾 / M倾覆，要求≥1.20
  - 抗隆起（Prandtl）：Ks = (γD·Nq + c·Nc) / (γ(H+D) + q)
  - 抗渗流：Kw = γ'·D / γw·Δh，要求≥1.50
- 监测预警体系：
  - 黄色预警：累计值达预警值70%
  - 橙色预警：累计值达预警值
  - 红色报警：达报警值或速率突增
- 应急预案：坍塌、涌水、建筑沉降处置流程
- 验收要点：分项验收标准、资料清单

**高支模详解（high-formwork.md）**：

- 荷载计算：
  - 永久荷载：模板、钢筋、混凝土、支架自重
  - 可变荷载：施工荷载（2.5kN/m²）、振捣荷载（2.0kN/m²）
  - 荷载组合：S = 1.35G + 1.4Q
- 立杆稳定性验算：
  - 计算长度：l0 = k × μ × h = 1.155 × 1.5 × h
  - 长细比：λ = l0 / i（φ48.3钢管 i=15.9mm）
  - 稳定系数φ：查表确定
  - 验算：N / (φ·A) ≤ f（f=205N/mm²）
  - 安全系数：K ≥ 2.0
- 剪刀撑设置：竖向+水平，间距6-8m
- 支架预压：1.1倍设计荷载，分级加载
- 拆除条件：强度要求表（按跨度分类）

**桥梁工程详解（bridge-construction.md）**：

- 桥型分类：梁桥、拱桥、斜拉桥、悬索桥、刚构桥
- 下部结构：
  - 钻孔灌注桩：工艺流程、质量控制（桩位≤50mm，垂直度≤1%，沉渣≤100mm）
  - 承台：大体积混凝土温控（分层≤300mm，温差≤25℃）
  - 墩柱：模板体系选择（组合模/大钢模/翻模/滑模/爬模）
  - 盖梁：支架形式（落地/抱箍/托架）、预应力张拉
- 上部结构：
  - 预制梁：预制场布置、施工流程、预应力张拉、存梁运输、架设方法
  - 现浇箱梁：支架方案、预压、浇筑顺序
  - 悬臂浇筑：挂篮设计、悬浇流程、线形控制、合龙施工
- 预应力施工：双控指标（张拉力+伸长量≤±6%）
- 成桥检测：静载试验、动载试验、外观检查

#### 1.6 使用示例

```
用户：帮我写一个深基坑施工方案，开挖深度12米，地质是淤泥质土，周边有住宅楼

AI执行流程：
1. 判断：12m > 8m，属于超危大工程，需专家论证
2. 调用：deep-excavation.md详解文档
3. 推荐：地下连续墙+多道内支撑
4. 输出：
   - 方案框架（8章节）
   - 分层开挖参数（每层≤2m，分段≤20m软土）
   - 稳定性验算计算书（公式+参数+过程）
   - 监测方案（预警值表）
   - 应急预案
   - 专家论证提示
```

### 2. project-manager（项目管理助手）

#### 2.1 功能概述

覆盖项目全生命周期管理，包括进度管控、成本管控、质量管理、风险管理、变更管理等核心职能。

#### 2.3 核心能力

| 能力模块 | 说明 |
|---------|------|
| WBS分解 | 5级结构分解框架 |
| 挣值法分析 | EVM指标计算（SV/CV/SPI/CPI） |
| 进度偏差应对 | 5种策略：增资源、调工序、优化方法、压缩工期、协商顺延 |
| 变更管理 | 变更流程、签证原则 |
| 报告编制 | 周报、月报、专题报告模板 |

#### 2.4 项目阶段识别

| 阶段 | 核心任务 | 主要输出 |
|------|---------|---------|
| 策划阶段 | WBS分解、计划编制 | 项目管理计划 |
| 实施阶段 | 进度跟踪、偏差分析 | 周报/月报 |
| 收尾阶段 | 竣工验收、结算 | 竣工报告 |

#### 2.5 挣值法（EVM）应用

**核心公式**：

```
SV（进度偏差） = EV - PV
  负值表示进度滞后

CV（成本偏差） = EV - AC
  负值表示成本超支

SPI（进度绩效指数） = EV / PV
  <1 表示滞后，>1 表示提前

CPI（成本绩效指数） = EV / AC
  <1 表示超支，>1 表示节约

EAC（完工估算） = BAC / CPI
```

#### 2.6 变更管理流程

```
变更发生 → 变更识别与记录 → 影响评估（工期/费用）
→ 变更申请（书面） → 审批（监理/业主）
→ 变更实施 → 变更结算
```

**关键原则**：先签证后施工，口头指令书面确认

#### 2.7 报告模板

**周报模板**：
- 本周完成工作（形象进度、工程量、质量、安全）
- 下周工作计划
- 存在问题及建议
- 需协调事项

**月报模板**：
- 工程概况
- 本月完成情况（进度偏差分析）
- 质量管理
- 安全文明施工
- 成本情况
- 下月计划

**专题报告模板**（进度滞后）：
- 进度现状与偏差
- 滞后原因分析（客观/主观）
- 影响评估
- 追赶措施
- 调整后计划

### 4. contract-reviewer（合同审查助手）

#### 4.1 功能概述

对建设工程合同进行专业审查，识别风险条款、不平等条款、缺失条款，提供修改建议和谈判策略。

#### 4.2 触发关键词

- "帮我审查合同"
- "看看这个合同有没有问题"
- "合同条款怎么改"
- "合同谈判策略"

#### 4.3 审查流程

```
接收合同文本 → 识别合同类型 → 系统性审查
→ 风险分级标注 → 修改建议 → 谈判策略
```

#### 4.4 八大审查维度

**1. 主体资格**
- 双方主体资格是否合法
- 签约主体与履约主体是否一致
- 代理人授权是否充分

**2. 合同价款与支付**
- 计价方式（固定总价/固定单价/成本加酬金）
- 付款节点、比例是否合理
- 预付款、进度款、结算款条款
- 质保金比例（≤5%）和返还时间
- 价格调整机制

**3. 工期条款**
- 开工日期、竣工日期是否明确
- 工期顺延条件是否完整
- 工期延误违约金是否合理
- 工期奖励条款

**4. 质量条款**
- 质量标准是否明确
- 质量保修期限（不低于法定最低）
- 质量缺陷责任划分

**5. 变更与索赔**
- 变更程序是否明确
- 变更计价原则
- 索赔时效（28天标准）
- 索赔程序

**6. 违约责任**
- 甲乙双方违约责任是否对等
- 违约金是否过高（超过损失30%可调整）
- 解除合同条件

**7. 争议解决**
- 争议解决方式（协商/调解/仲裁/诉讼）
- 仲裁机构或管辖法院是否明确

**8. 其他风险条款**
- 不可抗力范围
- 保险要求
- 知识产权归属
- 保密条款

#### 4.5 风险分级标准

| 级别 | 标识 | 说明 | 处理建议 |
|------|------|------|---------|
| 高风险 | 🔴 | 可能导致重大损失 | 必须修改 |
| 中风险 | 🟡 | 存在不利条款 | 建议修改 |
| 低风险 | 🟢 | 轻微瑕疵 | 可接受但建议完善 |
| 缺失项 | ⬜ | 重要条款缺失 | 需补充 |

#### 4.6 常见不平等条款库

**付款类**：
| 不平等条款 | 风险 | 建议 |
|-----------|------|------|
| "审计时间不限" | 🔴 结算遥遥无期 | 约定审计时限（6个月） |
| "业主有权扣留任何款项" | 🔴 付款无保障 | 明确扣款条件和上限 |
| "进度款以业主资金到位为前提" | 🔴 付款不确定 | 删除，约定固定付款日期 |

**工期类**：
| 不平等条款 | 风险 | 建议 |
|-----------|------|------|
| "工期延误每日罚款合同额1%" | 🔴 违约金过高 | 改为0.05%-0.1%，设总额上限 |
| "不论何种原因工期延误均承担责任" | 🔴 责任倒置 | 明确甲方原因、不可抗力免责 |

**变更索赔类**：
| 不平等条款 | 风险 | 建议 |
|-----------|------|------|
| "索赔须7日内提出否则视为放弃" | 🟡 时限过短 | 改为28天（FIDIC标准） |
| "承包方不得就设计变更提出额外费用" | 🔴 变更无补偿 | 删除，明确变更计价原则 |

#### 5.1 功能概述

```
事实梳理 → 法律关系识别 → 证据评估

```

#### 5.4 纠纷类型与法律依据

| 纠纷类型 | 法律依据 | 诉讼时效 |
|---------|---------|---------|
| 工程款纠纷 | 《民法典》第807条、建工司法解释 | 3年 |
| 工期延误 | 合同约定 + 《民法典》违约责任 | 3年 |
| 质量缺陷 | 《建筑法》第80条、《民法典》第803条 | 1年（质保期内）|
| 优先受偿权 | 《民法典》第807条 | 18个月 |
| 劳务纠纷 | 《劳动法》《劳动合同法》 | 1年 |

#### 5.5 证据强度分级

| 证据类型 | 强度 | 说明 |
|---------|------|------|
| 书面合同（盖章签字） | ⭐⭐⭐⭐⭐ | 最强 |
| 竣工验收报告 | ⭐⭐⭐⭐⭐ | 工程款请求权基础 |
| 书面签证/变更单 | ⭐⭐⭐⭐ | 变更索赔依据 |
| 往来函件（书面） | ⭐⭐⭐⭐ | 意思表示证明 |
| 微信/邮件记录 | ⭐⭐⭐ | 需完整截图+公证 |
| 录音录像 | ⭐⭐⭐ | 需合法取得 |
| 证人证言 | ⭐⭐ | 辅助证据 |
| 口头承诺 | ⭐ | 几乎无效 |

#### 5.6 胜诉概率评估

**评估维度**：
```
胜诉概率 = f(
  法律依据强度,
  证据充分程度,
  对方抗辩强度,
  司法实践倾向,
  执行可能性
)
```

**概率区间**：
| 概率 | 说明 | 建议 |
|------|------|------|
| >80% | 胜算较大 | 积极起诉，同时保全财产 |
| 60-80% | 有一定把握 | 起诉同时积极和解 |
| 40-60% | 胜负难料 | 优先和解，诉讼作为备选 |
| <40% | 胜算较小 | 重点考虑和解或调解 |

#### 5.7 诉讼路径选择

**选择仲裁**：
- 合同有仲裁条款
- 需要保密
- 希望快速解决（6-9个月）
- 涉及技术性争议

**选择诉讼**：
- 无仲裁条款
- 需要财产保全
- 对方有明确财产
- 案件事实清晰

#### 5.8 诉前必做清单

- [ ] 固定证据（公证、截图、备份）
- [ ] 申请财产保全
- [ ] 发送催告函（书面，留存送达证明）
- [ ] 行使优先受偿权（18个月内）
- [ ] 评估对方偿债能力

#### 5.9 优先受偿权要点

- **法律依据**：《民法典》第807条
- **行使期限**：自竣工之日或合同约定竣工之日起18个月内
- **行使方式**：协议折价 / 申请法院拍卖
- **优先顺序**：优先于抵押权和普通债权

#### 6.5 工程量计算规则

**土石方工程**：
- 挖基坑：体积计算，含工作面和放坡
- 放坡系数：一类土1:0.75，二类土1:0.5，三类土1:0.33
- 回填：挖方体积 - 基础体积

**混凝土工程**：
- 基础：按图示尺寸以体积计算

- 板：板面积×板厚（不扣除≤0.3m²孔洞）

**钢筋工程**：
```
钢筋重量(kg) = 钢筋长度(m) × 理论重量(kg/m)

常用理论重量：
φ6: 0.222, φ8: 0.395, φ10: 0.617

φ18: 2.00, φ20: 2.47, φ22: 2.98, φ25: 3.85
```

#### 7.5 风险评估（LEC法）

```
风险值 D = L × E × C

L（事故可能性）：0.1-10
E（人员暴露频率）：0.5-10
C（后果严重性）：1-100

D值分级：
>320  极高风险（红色）→ 立即停工
160-320  高风险（橙色）→ 限期整改
70-160  中等风险（黄色）→ 制定措施
20-70  低风险（蓝色）→ 注意
<20  可接受（绿色）→ 记录
```

- 特殊节点详图完整性

**给排水专业**：
- 给水系统分区（每区≤50m）
- 消防系统覆盖完整性
- 排水坡度标注
- 管道穿墙套管设置
- 卫生间排水立管位置

**暖通专业**：
- 风管截面与建筑净高协调
- 机房面积是否足够
- 防排烟系统覆盖
- 风口位置与装修协调

**电气专业**：
- 变配电室位置与负荷中心
- 配电系统图与平面图一致
- 消防电源配置
- 防雷接地系统
- 桥架路由与其他专业冲突

#### 10.6 BIM LOD等级

| LOD等级 | 应用阶段 | 模型精度 |
|---------|---------|---------|
| LOD 100 | 方案设计 | 体量模型 |
| LOD 200 | 初步设计 | 近似几何 |
| LOD 300 | 施工图设计 | 精确几何 |
| LOD 350 | 施工深化 | 含连接件 |
| LOD 400 | 施工安装 | 制造级精度 |
| LOD 500 | 竣工交付 | 竣工实际状态 |

#### 10.7 施工阶段BIM应用点

**高价值应用**：
- 碰撞检查（机电综合管线）
- 施工方案模拟（复杂节点/危大工程）
- 工程量提取与复核
- 4D进度模拟

#### 10.8 机电综合管线排布原则

```
优先级（从高到低）：
1. 消防管道（不可移动）
2. 重力排水管（坡度要求）
3. 通风管道（截面大，优先排布）
4. 给水/空调水管
5. 电缆桥架
6. 弱电桥架

净高控制：
- 走廊：≥2.2m
- 机房：≥2.5m
- 地下车库：≥2.2m（行车道≥2.4m）
```

### 4.2 典型协同示例

```
1. procurement-advisor：分析招标文件，制定投标策略
2. cost-estimator：估算造价，编制报价
3. construction-plan：编写技术标施工方案
4. contract-reviewer：中标后审查合同条款
```

**示例2：工程款纠纷处理**

```
1. litigation-risk：评估胜诉概率，制定诉讼策略
2. contract-reviewer：分析合同付款条款，识别违约
3. cost-estimator：核对工程量清单，计算应付款项
4. engineering-consultant：如有质量问题争议，提供技术分析
```

**示例3：危大工程专项**

```
1. construction-plan：编制深基坑专项方案
2. safety-inspector：制定安全监测方案，隐患排查
3. quality-controller：验收标准，检测计划
4. project-manager：进度管控，变更管理
```

## 五、使用最佳实践

### 5.1 信息提供建议

为获得更精准的支持，建议提供以下信息：

**施工方案类**：
- 工程名称、类型、规模
- 地质条件、周边环境
- 工期要求、质量等级
- 特殊技术要求

**合同审查类**：
- 合同全文或关键条款
- 双方主体信息
- 工程概况
- 特殊约定

**诉讼评估类**：
- 合同文本
- 争议焦点
- 现有证据清单
- 对方立场

**造价估算类**：
- 工程类型、规模
- 设计图纸或工程量清单
- 地区信息

### 5.2 输出格式选择

各Skill支持多种输出格式：

| 格式 | 适用场景 | 调用方式 |
|------|---------|---------|
| Markdown | 对话展示、快速查看 | 默认输出 |
| Word（.docx） | 正式文档、方案提交 | 调用 docx skill |
| Excel（.xlsx） | 清单、表格、数据 | 调用 xlsx skill |
| PPT（.pptx） | 汇报、演示 | 调用 pptx skill |

### 5.3 专业边界说明

各Skill提供的意见均为参考性质：

> ⚠️ 本技能提供的分析、建议仅供参考，不替代专业机构出具的正式意见。
> 涉及重大决策（结构安全、超危大工程、重大合同、诉讼案件），
> 建议委托具有相应资质的专业单位出具正式意见。

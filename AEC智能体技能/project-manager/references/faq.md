# FAQ

## 常见问题

### Q1: 项目经理的核心职责是什么？

**A**：项目经理核心职责：
1. **项目策划**：制定项目目标和计划
2. **团队管理**：组建和管理项目团队
3. **进度控制**：确保项目按期完成
4. **成本控制**：控制项目成本在预算内
5. **质量管理**：确保项目质量达标
6. **安全管理**：保障项目安全生产
7. **沟通协调**：协调各方关系

### Q2: 如何有效进行进度管控？

**A**：进度管控要点：
1. 编制详细的进度计划
2. 分解任务，明确责任
3. 定期检查进度执行情况
4. 及时识别偏差并分析原因
5. 采取纠偏措施
6. 关键线路重点管控

### Q3: 项目风险如何管理？

**A**：风险管理流程：
1. **风险识别**：识别潜在风险
2. **风险评估**：评估风险发生概率和影响
3. **风险应对**：制定应对措施
4. **风险监控**：跟踪风险变化
5. **风险报告**：定期汇报风险状态

### Q4: 如何提升团队凝聚力？

**A**：团队建设建议：
1. 明确团队目标和规则
2. 建立有效的沟通机制
3. 合理分配工作任务
4. 设立激励机制
5. 开展团队活动
6. 及时表彰优秀成员

### Q5: 如何处理项目变更？

**A**：变更处理流程：
1. 收集变更信息
2. 评估变更影响（工期、成本、质量）
3. 报请审批
4. 实施变更
5. 更新项目计划
6. 记录变更内容

### Q6: 项目收尾工作有哪些？

**A**：项目收尾工作：
1. 竣工验收
2. 结算办理
3. 资料归档
4. 质量保修
5. 项目总结
6. 人员解散
7. 合同关闭

## 疑难解答

| 问题 | 解决方案 |
|------|----------|
| 进度滞后 | 分析原因，增加资源，优化方案 |
| 成本超支 | 优化方案，减少浪费，加强签证管理 |
| 质量缺陷 | 返工整改，加强过程控制 |
| 团队冲突 | 沟通协调，明确职责，公平处理 |

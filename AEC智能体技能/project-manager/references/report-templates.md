# 项目报告模板库

## 工程周报模板

```
【工程周报】第XX期
项目名称：
报告周期：XXXX年XX月XX日 - XX月XX日
编制人：          审核人：

一、本周完成工作
1. 形象进度：[描述各区域/楼层完成情况]
2. 主要工程量：
   | 工序 | 计划量 | 完成量 | 完成率 |
   |------|--------|--------|--------|
3. 质量情况：[质量检查结果，问题整改情况]
4. 安全情况：[安全检查，隐患排查情况]

二、下周工作计划
1. 计划完成工程量：
2. 关键节点：
3. 资源安排：

三、存在问题及建议
1. 问题：[描述]
   建议：[解决方案]

四、需业主/监理协调事项
```

## 工程月报模板

```
【工程月报】XXXX年XX月
项目名称：
报告周期：XX月XX日 - XX月XX日

一、工程概况
二、本月完成情况
  2.1 形象进度
  2.2 工程量完成情况（附表）
  2.3 进度偏差分析
三、质量管理
  3.1 质量检查情况
  3.2 质量问题及整改
四、安全文明施工
五、成本情况（如适用）
六、下月工作计划
七、存在问题及建议
```

# Calculation Guide

## 项目管理指标计算

### 1. 进度指标

#### 1.1 进度偏差（SV）
**公式**: SV = BCWP - BCWS

其中：
- SV：进度偏差
- BCWP：已完成工作预算成本（Earned Value）
- BCWS：计划工作预算成本

**判断**：
- SV > 0：进度超前
- SV = 0：进度正常
- SV < 0：进度滞后

#### 1.2 进度绩效指数（SPI）
**公式**: SPI = BCWP / BCWS

**判断**：
- SPI > 1：进度超前
- SPI = 1：进度正常
- SPI < 1：进度滞后

---

### 2. 成本指标

#### 2.1 成本偏差（CV）
**公式**: CV = BCWP - ACWP

其中：
- CV：成本偏差
- ACWP：已完成工作实际成本

**判断**：
- CV > 0：成本节约
- CV = 0：成本正常
- CV < 0：成本超支

#### 2.2 成本绩效指数（CPI）
**公式**: CPI = BCWP / ACWP

**判断**：
- CPI > 1：成本节约
- CPI = 1：成本正常
- CPI < 1：成本超支

---

### 3. 工期计算

#### 3.1 总工期
**公式**: 总工期 = 准备时间 + 施工时间 + 验收时间 + 移交时间

#### 3.2 关键路径
计算方法：找出项目网络图中时间最长的路径

#### 3.3 自由时差
**公式**: 自由时差 = 紧后工作最早开始时间 - 本工作最早完成时间

---

### 4. 资源计算

#### 4.1 人力资源需求
**公式**: 人力资源需求 = 工作量 / （工作效率 × 工作时间）

#### 4.2 机械设备需求
**公式**: 设备需求 = 工作量 / （设备产量 × 使用时间）

---

### 5. 绩效指标

#### 5.1 合同履约率
**公式**: 合同履约率 = 实际完成产值 / 计划完成产值 × 100%

#### 5.2 质量合格率
**公式**: 质量合格率 = 合格数量 / 检查数量 × 100%

#### 5.3 安全事故率
**公式**: 事故率 = 事故次数 / 实际工作工时 × 100%

---

## 项目管理工具

### 1. 挣值管理（EVM）
| 指标 | 英文 | 含义 |
|-----|------|------|
| PV | Planned Value | 计划值 |
| EV | Earned Value | 挣值 |
| AC | Actual Cost | 实际成本 |
| SV | Schedule Variance | 进度偏差 |
| CV | Cost Variance | 成本偏差 |
| SPI | Schedule Performance Index | 进度绩效指数 |
| CPI | Cost Performance Index | 成本绩效指数 |

### 2. 项目面临指数
**公式**: TCPI = (BAC - EV) / (BAC - AC)

- BAC：完工预算
- TCPI > 1：需要提高效率
- TCPI < 1：可以适当放松

---

## 注意事项

### 计算要点
1. 进度指标要定期计算和跟踪
2. 成本指标要与产值挂钩
3. 挣值管理需要完整的数据支撑

### 常见问题
1. 数据不准确导致指标失真
2. 偏差分析不深入
3. 纠偏措施不及时

### 改进建议
- 建立数据收集机制
- 定期进行指标分析
- 及时采取纠偏措施

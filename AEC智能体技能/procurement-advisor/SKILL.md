# 工程采购与招投标顾问

## 招标类型识别

| 招标类型 | 适用条件 | 主要法规 |
|---------|---------|---------|
| 公开招标 | 必须招标项目 | 《招标投标法》 |
| 邀请招标 | 特殊情况经批准 | 《招标投标法》 |
| 政府采购 | 政府资金采购 | 《政府采购法》 |
| 竞争性谈判 | 技术复杂/紧急 | 《政府采购法》 |
| 询价采购 | 标准货物/小额 | 《政府采购法》 |

## 必须招标范围（《必须招标的工程项目规定》）

- 施工单项合同估算价 **≥400万元**
- 重要设备、材料采购单项合同估算价 **≥200万元**
- 勘察、设计、监理等服务单项合同估算价 **≥100万元**
- 项目总投资额 **≥3000万元**

## 招标文件核心章节

```
第一章 招标公告/投标邀请书
第二章 投标人须知
  - 项目概况
  - 资金来源与落实情况
  - 招标范围
  - 投标人资格要求
  - 踏勘现场
  - 投标文件编制要求
  - 投标报价
  - 投标有效期
  - 投标保证金
  - 开标、评标、定标
第三章 评标办法
第四章 合同条款及格式
第五章 工程量清单
第六章 图纸
第七章 技术标准和要求
第八章 投标文件格式
```

## 评标办法设计

### 综合评估法（推荐用于技术复杂项目）

```
综合得分 = 商务标得分×W1 + 技术标得分×W2 + 价格标得分×W3

典型权重：
- 商务标（资质/业绩/信誉）：20-30%
- 技术标（方案/人员/设备）：30-40%
- 价格标：40-50%
```

### 经评审最低价法（适用于技术简单、标准化项目）

```
评标基准价 = 有效投标报价算术平均值 × 下浮系数
最低价中标（须通过技术评审）
```

## 投标策略建议

### 报价策略
- **不平衡报价**：早期工程量单价高，后期低；工程量可能增加的项目单价高
- **技术标差异化**：突出独特施工方案、类似业绩、项目团队
- **废标风险规避**：参见 [references/bid-risks.md](references/bid-risks.md)

### 投标文件编制要点
1. 严格响应招标文件，逐条对应
2. 资质证书、业绩证明材料完整
3. 报价计算无算术错误
4. 签字盖章齐全
5. 投标保证金按时足额提交

## 合规红线

- 禁止串通投标（刑事责任）
- 禁止弄虚作假（取消资格+列入黑名单）
- 禁止低于成本价投标
- 禁止转包（分包须经发包方同意）

# Calculation Guide

## 采购成本计算

### 1. 采购价格计算

#### 1.1 价格组成
**公式**: 采购价格 = 材料成本 + 运输费用 + 包装费用 + 税费 + 利润

**材料成本** = 原材料 + 加工费 + 损耗

**示例**：
```
钢材采购：
原材料成本：4500元/吨
加工费：200元/吨
损耗：3%
运输费：80元/吨
包装费：20元/吨
税费：13%（增值税）

材料成本 = (4500+200)×1.03 = 4841元/吨
采购价格 = 4841 + 80 + 20 + 4841×0.13 = 5501元/吨
```

---

### 2. 采购降本计算

#### 2.1 规模效应降本
**公式**: 降本金额 = (原单价 - 新单价) × 采购数量

**示例**：
```
原采购单价：5000元/吨
批量采购后单价：4500元/吨
采购量：10000吨
降本金额 = (5000-4500)×10000 = 500万元
降本率 = (5000-4500)/5000 = 10%
```

#### 2.2 减少中间环节降本
**公式**: 降本金额 = 差价 × 采购数量

**示例**：
```
厂家直销价：4000元/吨
经销商供货价：4500元/吨
差价：500元/吨
年采购量：5000吨
降本金额 = 500 × 5000 = 250万元
```

---

### 3. 供应商报价分析

#### 3.1 综合成本分析
**公式**: 总成本 = 采购价 + 物流成本 + 质量成本 + 持有成本

- **物流成本**：运输、仓储、装卸
- **质量成本**：检验、返工、废品
- **持有成本**：资金占用、仓储费用

#### 3.2 TCO（总拥有成本）
**公式**: TCO = 采购成本 + 使用成本 + 处置成本

- **使用成本**：维护、能耗、培训
- **处置成本**：报废、回收

---

### 4. 采购指标

#### 4.1 采购绩效指标
| 指标 | 计算方法 | 目标值 |
|-----|---------|-------|
| 采购成本降低率 | (原成本-现成本)/原成本 | ≥5% |
| 供应商准时交付率 | 准时交付次数/总交付次数 | ≥95% |
| 采购质量合格率 | 合格数量/总数量 | ≥98% |
| 供应商开发周期 | 从寻源到准入的时间 | ≤30天 |

#### 4.2 资金时间价值
**公式**: 现值 = 未来值 / (1 + r)^n

**示例**：
```
1年后付款100万，年利率5%
现值 = 100 / 1.05 = 95.24万
提前付款可节约4.76万
```

## 注意事项

### 成本控制要点
1. 整合需求，形成规模效应
2. 直接对接厂家，减少中间环节
3. 合理安排采购时间
4. 适当延长付款周期获取优惠

### 价格波动应对
- 锁定长期合同价格
- 建立价格预警机制
- 设定价格调整条款
- 合理储备

### 成本优化方向
- 设计优化，减少用量
- 工艺优化，提高效率
- 物流优化，降低运费
- 库存优化，减少占用

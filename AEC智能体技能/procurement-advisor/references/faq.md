# FAQ

## 常见问题

### Q1: 公开招标和邀请招标有什么区别？

**A**：区别：
- **公开招标**：面向所有合格供应商，公开性强，竞争充分
- **邀请招标**：面向特定供应商，竞争有限，效率高

适用：
- 公开招标：金额≥200万元，通用材料设备
- 邀请招标：金额<200万元，特殊专业设备

### Q2: 如何选择合适的采购方式？

**A**：采购方式选择原则：
- **金额**：200万以上必须公开招标
- **标的**：技术复杂或有特殊要求可邀请招标
- **紧急**：紧急项目可询价或单一来源
- **唯一**：唯一供应商需单一来源采购

### Q3: 供应商准入审核哪些内容？

**A**：供应商审核要点：
1. **企业资质**：营业执照、资质证书、安全许可证
2. **生产能力**：产能、设备、人员
3. **财务状况**：注册资本、资产负债、信用情况
4. **历史业绩**：类似项目业绩、客户评价
5. **服务体系**：售后服务响应时间

### Q4: 如何控制采购质量？

**A**：质量控制措施：
1. 明确质量标准和技术要求
2. 供应商考察和评审
3. 到货验收和检验
4. 质量问题处理和追溯
5. 供应商绩效考核

### Q5: 采购合同有哪些关键条款？

**A**：关键条款：
1. 产品规格和数量
2. 价格和付款方式
3. 交货时间和地点
4. 质量标准和验收
5. 违约责任
6. 争议解决

### Q6: 如何应对材料价格波动？

**A**：应对措施：
1. **长期合同**：锁定价格或约定价格调整机制
2. **价格预警**：建立价格监测机制
3. **合理储备**：适度增加安全库存
4. **替代方案**：寻找替代材料或供应商

## 疑难解答

| 问题 | 解决方案 |
|------|----------|
| 供应商交货延迟 | 催促发货，必要时索赔；更换供应商 |
| 质量不达标 | 退货或返工；要求供应商改进 |
| 价格高于预期 | 重新谈判，寻找替代供应商 |
| 付款争议 | 按合同约定，保留沟通记录 |

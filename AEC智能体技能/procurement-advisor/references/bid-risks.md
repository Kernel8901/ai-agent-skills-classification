# 投标废标风险清单

## 资格性废标（一票否决）

| 废标原因 | 预防措施 |
|---------|---------|
| 资质等级不满足要求 | 开标前核查资质有效期和等级 |
| 营业执照经营范围不含招标内容 | 核查营业执照范围 |
| 安全生产许可证过期 | 提前更新，确保有效 |
| 投标保证金金额不足或形式不符 | 严格按招标文件要求提交 |
| 投标保证金逾期到账 | 提前2-3天提交，留足时间 |
| 法定代表人授权委托书缺失 | 检查授权委托书格式和签章 |
| 联合体协议未附或不规范 | 联合体投标必须附协议 |

## 符合性废标（技术/商务）

| 废标原因 | 预防措施 |
|---------|---------|
| 投标文件未按要求密封 | 严格按招标文件密封要求操作 |
| 投标文件未签字或盖章 | 逐页检查签字盖章 |
| 报价超过最高限价 | 报价前核查最高限价 |
| 工期承诺不满足要求 | 明确承诺满足招标工期 |
| 质量标准承诺不满足 | 明确承诺满足质量要求 |
| 投标有效期不足 | 承诺有效期不少于招标要求 |
| 算术错误导致总价与单价不符 | 提交前仔细核算，单价优先 |

## 信用类废标

| 废标原因 | 预防措施 |
|---------|---------|
| 被列入黑名单 | 定期自查信用记录 |
| 近期有重大质量/安全事故 | 如实申报，评估影响 |
| 拖欠农民工工资记录 | 及时处理劳资纠纷 |

## 投标文件提交前检查清单

- [ ] 资质证书（原件扫描/复印件加盖公章）
- [ ] 营业执照副本
- [ ] 安全生产许可证
- [ ] 法定代表人身份证明
- [ ] 授权委托书（如非法定代表人签字）
- [ ] 投标函（签字盖章）
- [ ] 投标报价单（无算术错误）
- [ ] 工程量清单报价（每项均已填写）
- [ ] 施工组织设计/技术方案
- [ ] 项目经理资质证书及授权书
- [ ] 类似工程业绩证明
- [ ] 投标保证金缴纳凭证
- [ ] 密封完整，标注正确
- [ ] 份数满足要求（正本+副本）

# 工程造价估算与清单编制助手

## 造价阶段识别

| 阶段 | 精度 | 主要方法 | 输出 |
|------|------|---------|------|
| 投资估算 | ±30% | 指标法、类比法 | 估算表 |
| 设计概算 | ±15% | 概算定额法 | 概算书 |
| 施工图预算 | ±5% | 清单计价/定额计价 | 预算书 |
| 竣工结算 | 精确 | 实际工程量×合同单价 | 结算书 |

## 清单编制规范（GB 50500-2013）

### 清单项目编码规则
```
XX  XX  XX  XXX  XXX
专业 章  节  清单项 子目
```

### 分部分项工程量清单格式
```
| 序号 | 项目编码 | 项目名称 | 项目特征 | 计量单位 | 工程量 | 综合单价 | 合价 |
|------|---------|---------|---------|---------|--------|---------|------|
```

### 措施项目清单
**总价措施**（不计量）：安全文明施工费、夜间施工、二次搬运、冬雨季施工
**单价措施**（可计量）：脚手架、模板、垂直运输、超高施工增加

## 综合单价组成

```
综合单价 = 人工费 + 材料费 + 机械费 + 管理费 + 利润 + 风险费
```

### 定额套用原则
1. 优先套用与工程特征完全匹配的定额子目
2. 无完全匹配时，套用相近子目并换算
3. 缺项时编制补充定额，报发包方确认

## 结算审核要点

### 高频争议点
- **工程量偏差**：图纸工程量 vs 实际施工量，以竣工图为准
- **变更计价**：合同有约定按约定，无约定参照清单综合单价
- **材料调差**：合同约定的调差范围、基准价、调差系数
- **签证认定**：签证内容是否与合同范围重叠
- **索赔费用**：证据链是否完整，时效是否满足

## 输出格式

- 估算：汇总表 + 分项明细
- 清单：标准表格（可导出 Excel，调用 xlsx skill）
- 分析报告：造价构成饼图描述 + 文字分析

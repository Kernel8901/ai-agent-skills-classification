# Calculation Guide

## 建筑工程造价计算

### 1. 造价组成

#### 1.1 建设工程造价构成
**公式**: 总造价 = 建安工程费 + 设备及工器具购置费 + 工程建设其他费用 + 预备费 + 建设期利息

**建安工程费** = 人工费 + 材料费 + 施工机具使用费 + 企业管理费 + 利润 + 规费 + 税金

**示例**（住宅项目，单方造价3500元/平米）：
```
建安工程费：2800元/平米
  - 人工费：450元/平米（16%）
  - 材料费：1400元/平米（50%）
  - 机械费：140元/平米（5%）
  - 企业管理费：280元/平米（10%）
  - 利润：140元/平米（5%）
  - 规费：112元/平米（4%）
  - 税金：278元/平米（10%）

其他费用：700元/平米
  - 土地费：400元/平米
  - 配套费：150元/平米
  - 建设单位管理费：50元/平米
  - 监理费：30元/平米
  - 设计费：40元/平米
  - 预备费：30元/平米
```

### 2. 工程量计算

#### 2.1 建筑面积计算
**公式**: 建筑面积 = Σ(层建筑面积)

**注意事项**：
- 全面积：建筑各层水平投影面积
- 半面积：雨篷、挑廊、阳台按1/2面积计算
- 不计算面积：层高不足2.2m的夹层、管道井

#### 2.2 钢筋工程量
**公式**: 钢筋重量 = 钢筋长度 × 钢筋理论重量

**钢筋理论重量**：d²×0.00617（d为直径）

**示例**：
```
梁钢筋：φ20，10根，长度8m
重量 = 10×8×20²×0.00617 = 198kg
```

#### 2.3 混凝土工程量
**公式**: 混凝土体积 = 构件体积 - 钢筋体积

**简化**：混凝土体积 = 构件体积（钢筋体积约占1%-2%）

---

### 3. 费用计算

#### 3.1 企业管理费
**公式**: 企业管理费 = 取费基数 × 企业管理费费率

**取费基数**：人工费或人工费+机械费

**参考费率**（以人工费为基数）：
- 住宅：18%-25%
- 公建：20%-28%
- 市政：15%-22%

#### 3.2 利润
**公式**: 利润 = 取费基数 × 利润率

**参考利润率**：5%-12%，根据工程类别和企业确定

#### 3.3 规费
**公式**: 规费 = 取费基数 × 规费费率

**规费费率**：一般约4%-6%（社保费、住房公积金、工程排污费）

#### 3.4 税金
**公式**: 税金 = 税前造价 × 税率

**税率**：
- 一般纳税人：9%（简易征收3%）
- 小规模纳税人：3%

### 4. 造价指标

#### 4.1 常用造价指标

| 项目类型 | 造价指标（元/平米） | 钢筋含量（kg/平米） | 混凝土含量（m³/平米） |
|---------|------------------|-------------------|---------------------|
| 住宅（高层） | 3200-4000 | 45-65 | 0.35-0.45 |
| 住宅（多层） | 2500-3200 | 35-50 | 0.30-0.40 |
| 商业（高层） | 4000-6000 | 60-90 | 0.45-0.60 |
| 办公（高层） | 3500-5000 | 55-80 | 0.40-0.55 |
| 地下室 | 2500-3500 | 100-150 | 0.80-1.20 |

#### 4.2 安装工程指标
- 电气：150-300元/平米
- 给排水：80-150元/平米
- 通风空调：200-500元/平米
- 消防：100-200元/平米

## 注意事项

### 计算要点
1. 工程量计算必须严格按计算规则
2. 材料价格要符合市场实际
3. 费率取值要符合当地规定
4. 措施项目要考虑施工方案

### 常见问题
1. 钢筋含量超标：优化结构设计
2. 混凝土含量过高：优化截面尺寸
3. 变更签证计价争议：及时确认单价
4. 结算拖期：提前准备结算资料

### 计算软件
- 广联达GTJ：土建算量
- 广联达GCL：安装算量
- 鲁班土建：算量软件
- 品茗土建：算量软件
- 斯维尔：BIM算量

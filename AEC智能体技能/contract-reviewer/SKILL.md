# 工程合同审查助手

## 审查流程

```
接收合同文本 → 识别合同类型 → 系统性审查
→ 风险分级标注 → 修改建议 → 谈判策略
```

## 审查维度

### 必查项清单

**1. 主体资格**
- [ ] 双方主体资格是否合法（营业执照、资质证书）
- [ ] 签约主体与实际履约主体是否一致
- [ ] 代理人授权是否充分

**2. 合同价款与支付**
- [ ] 计价方式（固定总价/固定单价/成本加酬金）是否明确
- [ ] 付款节点、付款比例是否合理
- [ ] 预付款、进度款、结算款条款
- [ ] 质保金比例（不超过5%）和返还时间
- [ ] 价格调整机制（材料价格波动）

**3. 工期条款**
- [ ] 开工日期、竣工日期是否明确
- [ ] 工期顺延条件是否完整（甲方原因/不可抗力/设计变更）
- [ ] 工期延误违约金是否合理（上限约定）
- [ ] 工期奖励条款（如有）

**4. 质量条款**
- [ ] 质量标准是否明确（合格/优良/具体指标）
- [ ] 质量保修期限（不低于法定最低标准）
- [ ] 质量缺陷责任划分

**5. 变更与索赔**
- [ ] 变更程序是否明确
- [ ] 变更计价原则
- [ ] 索赔时效（28天/合理期限）
- [ ] 索赔程序

**6. 违约责任**
- [ ] 甲乙双方违约责任是否对等
- [ ] 违约金是否过高（超过损失30%可申请调整）
- [ ] 解除合同条件

**7. 争议解决**
- [ ] 争议解决方式（协商/调解/仲裁/诉讼）
- [ ] 仲裁机构或管辖法院是否明确
- [ ] 适用法律

**8. 其他风险条款**
- [ ] 不可抗力范围是否合理
- [ ] 保险要求
- [ ] 知识产权归属
- [ ] 保密条款

## 风险分级

| 级别 | 标识 | 说明 |
|------|------|------|
| 高风险 | 🔴 | 可能导致重大损失，必须修改 |
| 中风险 | 🟡 | 存在不利条款，建议修改 |
| 低风险 | 🟢 | 轻微瑕疵，可接受但建议完善 |
| 缺失项 | ⬜ | 重要条款缺失，需补充 |

```

**审查结论**：[建议签署/修改后签署/不建议签署]

### 高风险条款
[条款位置] 🔴
原文：...
问题：...
修改建议：...

### 中风险条款
...

### 缺失条款
...

### 谈判建议
优先争取：...
可以让步：...
底线条款：...
```

## 重要提示

> ⚠️ 本审查意见仅供参考，不构成法律意见。重大合同（金额大、风险高）建议委托专业律师出具正式法律意见书。

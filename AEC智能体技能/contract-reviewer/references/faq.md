# FAQ

## 常见问题

### Q1: 固定总价合同能否调整？

**A**: 视具体情况而定：
- 合同约定可以调价的：按约定调整
- 设计变更导致的工程量变化：可以调整
- 法律法规变化：应当调整
- 市场价格异常波动：可主张情事变更
- 合同明确约定不调整的：原则不可调整

### Q2: 工期延误如何取证？

**A**: 工期延误证据包括：
1. 监理日志：记录天气、停工原因
2. 会议纪要：各方确认的进度情况
3. 往来函件：发包人指令、承包人报告
4. 进度报告：实际完成工程量
5. 照片视频：现场施工情况
6. 气象资料：不利天气的证明

### Q3: 签证被拒绝怎么办？

**A**: 处理方式：
1. 检查签证程序是否符合合同约定
2. 补充完善签证资料
3. 发送书面催告函
4. 在约定的时效内提起索赔
5. 保留证据以便后续诉讼

### Q4: 承包人如何防范合同风险？

**A**: 风险防范要点：
1. 签约前充分审查合同条款
2. 完善合同交底，理解合同要点
3. 履约过程注意收集证据
4. 及时书面提出索赔和签证
5. 定期梳理合同履约情况

### Q5: 发包人拖延付款如何应对？

**A**: 应对措施：
1. 发送催款函，明确付款义务和期限
2. 依据合同约定主张逾期付款违约金
3. 评估是否具备暂停施工的条件
4. 及时行使合同解除权（如符合条件）
5. 诉诸法律途径追讨欠款

### Q6: 保修期缺陷如何处理？

**A**: 处理流程：
1. 现场勘查，确认缺陷情况
2. 分析原因，划分责任
3. 修复方案商定
4. 实施修复
5. 验收签字，留下记录

## 疑难解答

| 问题 | 解决方案 |
|------|----------|
| 合同解释有争议 | 按合同目的、交易习惯解释，兼顾公平 |
| 发包人拒签签证 | 留置送达相关函件，保留送达证据 |
| 分包人责任不清 | 依据总分包合同约定和法律规定划分 |
| 黑白合同效力 | 以备案的中标合同为准，但私下协议可能无效 |
| 实际施工人权益 | 直接向发包人主张，但以发包人欠付为限 |

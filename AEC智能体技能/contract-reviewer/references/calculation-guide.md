# Calculation Guide

## 合同价款计算

### 1. 工程量清单计价

#### 1.1 综合单价计算
**公式**: 综合单价 = 人工费 + 材料费 + 施工机具使用费 + 管理费 + 利润 + 一定范围内的风险费

其中：
- 人工费 = Σ(人工工日×人工单价)
- 材料费 = Σ(材料消耗量×材料单价)
- 施工机具使用费 = Σ(机械台班×台班单价)
- 管理费 = (人工费+材料费+施工机具使用费)×管理费费率
- 利润 = (人工费+材料费+施工机具使用费+管理费)×利润率

**示例**：
```
某清单项目：砌筑砖墙10m³
人工：20工日×100元/工日 = 2000元
材料：砖5200块×0.5元/块 = 2600元
砂浆：2m³×400元/m³ = 800元
机具：2台班×200元/台班 = 400元
管理费：(2000+2600+800+400)×10% = 580元
利润：(2000+2600+800+400+580)×8% = 510元
综合单价 = (2000+2600+800+400+580+510)/10 = 689元/m³
```

### 2. 工期延误索赔计算

#### 2.1 误期赔偿费计算
**公式**: 误期赔偿费 = 误期天数 × 合同约定的误期赔偿费金额

**示例**：
```
合同约定：误期赔偿费2000元/天
实际延误45天
误期赔偿 = 45 × 2000 = 90000元
```

#### 2.2 工期延误索赔计算
**公式**: 索赔金额 = 直接费用 + 间接费用 + 利润

- **直接费用**：人工费、材料费、机械使用费
- **间接费用**：管理费、规费
- **利润**：按合同约定计算

---

### 3. 价格调整计算

#### 3.1 价格指数调整法
**公式**: P = P0 × (a + b×F1/F0 + c×F2/F0 + ...)

其中：
- P: 调整后的工程价款
- P0: 合同约定工程价款
- a: 固定权重
- b、c: 各项费用权重
- F0: 基准价格指数
- F1: 当前价格指数

**示例**：
```
合同约定：固定权重a=0.15，人工b=0.45，材料c=0.40
基准指数：人工100，材料100
当前指数：人工115，材料110
P = P0 × (0.15 + 0.45×115/100 + 0.40×110/100)
  = P0 × (0.15 + 0.5175 + 0.44)
  = P0 × 1.1075
→ 价款上调10.75%
```

---

### 4. 变更价款计算

#### 4.1 合同有适用项目
**公式**: 变更价款 = 合同适用的综合单价 × 变更工程量

#### 4.2 合同无适用项目但有类似项目
**公式**: 变更价款 = 类似项目综合单价 × (1+调整系数)

#### 4.3 合同无适用或类似项目
**公式**: 变更价款 = 成本 + 利润 + 管理费

- 成本 = 人工费 + 材料费 + 施工机具使用费 + 措施费 + 规费

## 合同履约指标

### 1. 付款比例
- 预付款：一般为合同价款的10%-30%
- 进度款：按已完成工程量支付，一般为80%-90%
- 尾款：结算后支付，一般为5%-10%

### 2. 保修金
- 保修金比例：一般为合同价款的3%-5%
- 返还时间：一般为竣工验收后1-5年分批返还

### 3. 违约金上限
- 法定上限：不超过合同标的额的20%
- 司法实践中：一般不超过实际损失的130%

---

## 注意事项

### 计算要点
1. 计价规则要以合同约定为准
2. 工程量计算要依据图纸和工程量计算规则
3. 费率取值要符合当地造价管理部门规定
4. 变更计价要及时准确，避免累积争议

### 常见问题
1. 清单漏项：及时提出，按合同约定调整
2. 计价漏项：按合同约定或协商确定
3. 价格波动：按合同约定调整机制计算
4. 签证变更：及时确认，保留书面证据

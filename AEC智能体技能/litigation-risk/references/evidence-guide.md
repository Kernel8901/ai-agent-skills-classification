# 工程诉讼证据指南

## 工程款纠纷证据清单

### 必备证据
- [ ] 建设工程施工合同（含补充协议）
- [ ] 竣工验收报告/竣工验收备案表
- [ ] 结算书（双方确认版）
- [ ] 付款记录（银行流水/收据）
- [ ] 催款函及送达证明

### 补强证据
- [ ] 工程量签证单
- [ ] 监理日志/监理报告
- [ ] 往来函件（催款/回复）
- [ ] 微信/邮件催款记录（截图+公证）
- [ ] 对方认可欠款的书面文件

### 证据缺口补救
| 缺失证据 | 补救方式 |
|---------|---------|
| 无书面合同 | 用实际履行证明（付款记录、施工记录、验收记录） |
| 未竣工验收 | 证明对方已实际使用（擅自使用视为验收） |
| 结算未确认 | 申请司法鉴定确定工程价款 |
| 催款无回应 | 公证送达，或通过EMS快递留存签收记录 |

## 工期延误纠纷证据清单

### 证明己方无责
- [ ] 开工令/开工报告
- [ ] 甲方指令（导致停工/窝工的书面指令）
- [ ] 设计变更通知单
- [ ] 不可抗力证明（气象记录、政府文件）
- [ ] 工期顺延申请及审批记录
- [ ] 监理确认的停工记录

### 证明对方违约
- [ ] 合同约定的开工条件（场地、图纸、预付款）
- [ ] 对方未按时提供条件的证明
- [ ] 工期延误损失计算依据

## 质量纠纷证据清单

### 质量缺陷证明
- [ ] 质量检测报告（有资质机构出具）
- [ ] 现场照片/视频（时间戳）
- [ ] 监理质量整改通知单
- [ ] 质量问题书面通知及对方回复

### 损失证明
- [ ] 修复费用报价/决算
- [ ] 因质量问题导致的停产损失证明
- [ ] 第三方损失赔偿记录

## 财产保全指引

### 可申请保全的财产
- 银行账户（需提供开户行信息）
- 应收账款（对方在其他项目的工程款）
- 不动产（房产、土地）
- 车辆、设备

### 保全申请要点
- 提供担保（一般为保全金额的30%，可用保险代替）
- 提供被申请人财产线索
- 紧急情况可申请诉前保全（48小时内裁定）

## 优先受偿权行使

**法律依据**：《民法典》第807条
**行使期限**：自建设工程竣工之日或合同约定竣工之日起18个月内
**行使方式**：
1. 与发包人协议折价
2. 申请法院拍卖工程，就拍卖价款优先受偿

**注意**：优先受偿权优先于抵押权和普通债权，但需在18个月内主张，否则消灭。

# FAQ

## 常见问题

### Q1: 建设工程纠纷如何选择管辖法院？

**A**：建设工程纠纷管辖法院选择：
1. **协议约定**：优先选择约定的法院
2. **合同履行地**：工程所在地法院
3. **被告住所地**：被告注册地法院
4. **级别管辖**：标的金额决定级别

注意：不动产纠纷由不动产所在地法院专属管辖

### Q2: 诉讼时效如何计算？

**A**：建设工程纠纷诉讼时效：
1. **工程款**：3年，从应付工程款之日起算
2. **质量保证金**：2年，从保证金返还之日起算
3. **优先受偿权**：6个月，从工程折价或拍卖之日起算
4. **侵权纠纷**：3年，从知道或应当知道侵权之日起算

诉讼时效可中断：主张权利、对方同意、提起诉讼

### Q3: 建设工程优先受偿权如何行使？

**A**：建设工程优先受偿权：
1. **行使条件**：承包人已按约定施工
2. **行使方式**：诉讼或申请拍卖
3. **优先范围**：工程款本金、利息、违约金
4. **限制**：不得对抗已支付全部或大部分房款的买受人

### Q4: 如何防范工程款风险？

**A**：工程款风险防范：
1. 签约前审查发包人资信
2. 合同约定明确付款条件和时间
3. 及时申报已完成工程量
4. 及时主张工程款，避免时效过期
5. 约定担保或保函

### Q5: 发生安全事故如何处理？

**A**：安全事故处理流程：
1. 立即启动应急预案
2. 抢救伤员，保护现场
3. 报告相关部门
4. 配合事故调查
5. 妥善处理赔偿
6. 总结教训，整改到位

## 疑难解答

| 问题 | 解决方案 |
|------|----------|
| 发包人拖延结算 | 按合同约定主张结算权利，必要时诉讼 |
| 质保金难以返还 | 按合同约定主张返还，质保期届满即主张 |
| 对方转移财产 | 申请财产保全，冻结对方账户 |
| 判决难以执行 | 申请强制执行，查询对方财产线索 |

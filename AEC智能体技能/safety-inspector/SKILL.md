# 工程安全隐患排查助手

## 检查模式

| 检查类型 | 触发词 | 输出 |
|---------|--------|------|
| 综合安全检查 | "全面检查"、"安全巡查" | JGJ 59评分表 |
| 专项检查 | "脚手架检查"、"基坑检查" | 专项检查表 |
| 隐患整改 | "整改通知"、"隐患处理" | 整改通知单 |
| 风险评估 | "危险源辨识"、"风险分级" | 风险清单 |
| 应急预案 | "应急预案"、"事故处置" | 预案文本 |

## 安全检查评分（JGJ 59-2011）

### 十项检查内容及权重

| 检查项目 | 满分 | 保证项目 |
|---------|------|---------|
| 安全管理 | 10 | 否 |
| 文明施工 | 20 | 否 |
| 脚手架 | 10 | 是 |
| 基坑工程 | 10 | 是 |
| 模板支架 | 10 | 是 |
| 高处作业 | 10 | 是 |
| 施工用电 | 10 | 是 |
| 物料提升机与施工升降机 | 10 | 是 |
| 塔式起重机与起重吊装 | 10 | 是 |
| 施工机具 | 10 | 否 |

**评定标准**：
- 优良：≥80分，且保证项目无零分
- 合格：≥70分，且保证项目无零分
- 不合格：<70分，或保证项目有零分

## 风险评估（LEC法）

```
风险值 D = L × E × C

L（事故发生可能性）：0.1-10
E（人员暴露频率）：0.5-10
C（事故后果严重性）：1-100

D值分级：
>320  极高风险（红色）→ 立即停工整改
160-320  高风险（橙色）→ 限期整改
70-160  中等风险（黄色）→ 制定措施
20-70  低风险（蓝色）→ 注意
<20  可接受风险（绿色）→ 记录
```

## 应急预案框架

```
1. 总则（目的、适用范围、编制依据）
2. 工程概况及危险源分析
3. 应急组织机构与职责
4. 预防与预警
   - 危险源监控
   - 预警条件与发布
5. 应急响应
   - 响应分级（一级/二级/三级）
   - 各类事故处置程序
     · 高处坠落
     · 物体打击
     · 坍塌事故
     · 触电事故
     · 火灾事故
     · 机械伤害
6. 应急保障（人员/物资/通信）
7. 培训与演练
8. 附件（联系方式/医院位置/疏散路线图）
```

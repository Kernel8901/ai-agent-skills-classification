# 施工现场高频安全隐患清单

## 脚手架隐患

| 隐患描述 | 风险等级 | 整改措施 |
|---------|---------|---------|
| 立杆基础不平整，无垫板/底座 | 🔴重大 | 立即整改，加设垫板，夯实基础 |
| 连墙件间距超规范（>3步3跨） | 🔴重大 | 补设连墙件，不得超过规范要求 |
| 剪刀撑缺失或设置不规范 | 🟡一般 | 按规范补设剪刀撑 |
| 脚手板铺设不满铺，有探头板 | 🟡一般 | 满铺脚手板，探头板不超过150mm |
| 防护栏杆高度不足（<1.2m） | 🟡一般 | 补设防护栏杆至规范高度 |
| 无密目安全网或破损 | 🟡一般 | 补设或更换密目安全网 |
| 架体与建筑物间隙过大（>150mm） | 🟡一般 | 加设翻板或防护措施 |

## 基坑工程隐患

| 隐患描述 | 风险等级 | 整改措施 |
|---------|---------|---------|
| 坑边堆载超过设计荷载 | 🔴重大 | 立即清除超载，设置警戒线 |
| 支护结构变形超预警值 | 🔴重大 | 立即停工，启动应急预案 |
| 坑边无防护栏杆 | 🔴重大 | 设置1.2m高防护栏杆+警示标志 |
| 降水井未正常运行 | 🔴重大 | 立即修复，防止坑底隆起 |
| 坑壁渗水、流砂 | 🔴重大 | 立即处理，必要时停工 |
| 监测点被破坏 | 🟡一般 | 立即恢复监测点，补测数据 |
| 坑边无排水沟 | 🟡一般 | 设置截水沟，防止地表水入坑 |

## 高处作业隐患

| 隐患描述 | 风险等级 | 整改措施 |
|---------|---------|---------|
| 临边无防护（楼层边、屋面边） | 🔴重大 | 立即设置防护栏杆或安全网 |
| 洞口无防护（楼板洞、电梯井） | 🔴重大 | 加设盖板或防护栏杆 |
| 作业人员未系安全带 | 🔴重大 | 立即停止作业，系好安全带 |
| 安全带无可靠挂点 | 🔴重大 | 设置安全绳或生命线 |
| 攀登作业无安全措施 | 🟡一般 | 设置固定爬梯或安全绳 |

## 施工用电隐患

| 隐患描述 | 风险等级 | 整改措施 |
|---------|---------|---------|
| 配电箱无漏电保护器 | 🔴重大 | 立即安装漏电保护器 |
| 电缆线破损、裸露 | 🔴重大 | 立即更换或修复绝缘 |
| 未执行TN-S系统（三级配电两级保护） | 🔴重大 | 按规范整改配电系统 |
| 配电箱无门、无锁 | 🟡一般 | 安装箱门并上锁 |
| 电缆线拖地、浸水 | 🟡一般 | 架空或穿管保护 |
| 照明与动力混用同一回路 | 🟡一般 | 分开设置照明回路 |

## 起重吊装隐患

| 隐患描述 | 风险等级 | 整改措施 |
|---------|---------|---------|
| 塔吊无有效检验合格证 | 🔴重大 | 立即停用，重新检验 |
| 吊索具破损、超期使用 | 🔴重大 | 立即更换合格吊索具 |
| 超载作业 | 🔴重大 | 立即停止，严格按额定载荷作业 |
| 吊装区域无警戒 | 🔴重大 | 设置警戒区，禁止人员进入 |
| 司机无证操作 | 🔴重大 | 立即停止，持证人员操作 |
| 限位装置失效 | 🔴重大 | 立即修复后方可使用 |

## 消防安全隐患

| 隐患描述 | 风险等级 | 整改措施 |
|---------|---------|---------|
| 动火作业无审批 | 🔴重大 | 立即停止，办理动火证 |
| 易燃材料堆放在动火区域附近 | 🔴重大 | 清除易燃物，保持安全距离 |
| 消防器材配置不足或过期 | 🟡一般 | 按规范配置，更换过期器材 |
| 疏散通道堵塞 | 🟡一般 | 立即清理，保持畅通 |
| 宿舍使用大功率电器 | 🟡一般 | 没收违规电器，加强教育 |

# FAQ

## 常见问题

### Q1: 危大工程包括哪些？

**A**：危险性较大的分部分项工程包括：
1. 基坑工程：深度≥3m
2. 模板工程：高度≥8m或跨度≥18m
3. 起重吊装：非常规起重设备
4. 脚手架：高度≥24m
5. 拆除工程：可能影响行人安全
6. 其它：隧道工程、爆破工程等

超过一定规模的危大工程需专家论证。

### Q2: 三级安全教育包括哪些？

**A**：三级安全教育：
1. **公司级**：安全生产法律法规、规章制度
2. **项目级**：项目安全管理制度、注意事项
3. **班组级**：岗位安全操作规程、应急处理

教育要有记录，考核合格后方可上岗。

### Q3: 安全检查有哪些类型？

**A**：安全检查类型：
1. **定期检查**：周检、月检、季检
2. **专项检查**：针对特定危险源
3. **日常巡查**：日常巡检
4. **季节性检查**：针对季节特点
5. **综合性检查**：全面检查

### Q4: 隐患如何分类？

**A**：隐患分类：
1. **按严重程度**：
   - 一般隐患：现场立即整改
   - 重大隐患：停工整改
2. **按专业**：机械、电气、消防等
3. **按部位**：基坑、脚手架、临电等

### Q5: 应急响应程序是什么？

**A**：应急响应：
1. **事故发生后**：立即启动应急预案
2. **现场处置**：抢救伤员，保护现场
3. **报告程序**：按规定向上级报告
4. **应急救援**：专业救援队伍到场前进行自救
5. **事后处理**：配合调查，处理善后

### Q6: 特种作业人员有哪些？

**A**：特种作业人员：
1. 电工
2. 焊工
3. 起重机械司机
4. 塔吊司机
5. 架子工
6. 高处作业人员

特种作业人员必须持证上岗。

## 疑难解答

| 问题 | 解决方案 |
|------|----------|
| 临边防护不到位 | 立即安装防护栏杆 |
| 临时用电不规范 | 重新接线，整改配电箱 |
| 脚手架不稳 | 重新搭设，加固基础 |
| 安全带未使用 | 停工教育，严肃处理 |

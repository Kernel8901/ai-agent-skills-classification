# Calculation Guide

## 安全指标计算

### 1. 事故指标

#### 1.1 事故频率
**公式**: 事故频率 = (事故次数 / 实际工作工日) × 1000‰

**示例**：
```
某工地100人，工作30天，发生轻伤事故1起
事故频率 = 1 / (100×30) × 1000 = 0.33‰
```

#### 1.2 事故严重率
**公式**: 严重率 = (损失工日 / 实际工作工日) × 1000‰

---

### 2. 安全检查指标

#### 2.1 隐患整改率
**公式**: 整改率 = (已整改隐患数 / 隐患总数) × 100%

#### 2.2 隐患按期整改率
**公式**: 按期整改率 = (按期整改数 / 应整改数) × 100%

#### 2.3 检查覆盖率
**公式**: 覆盖率 = (实际检查次数 / 计划检查次数) × 100%

---

### 3. 安全评价指标

#### 3.1 安全生产条件评分
**公式**: 得分 = Σ(分项得分 × 权重)

| 分项 | 权重 |
|-----|------|
| 安全管理 | 0.15 |
| 文明施工 | 0.10 |
| 脚手架 | 0.10 |
| 基坑支护 | 0.10 |
| 模板工程 | 0.10 |
| 三宝四口 | 0.15 |
| 施工用电 | 0.10 |
| 外用电梯 | 0.10 |
| 塔吊 | 0.10 |

---

### 4. 防护设施计算

#### 4.1 脚手架承载计算
**公式**: N ≤ φAf

其中：
- N：立杆轴向力
- φ：稳定系数
- A：截面积
- f：钢材设计强度

#### 4.2 临边防护栏杆荷载
**公式**: F = q × L

其中：
- F：栏杆承受的力
- q：水平荷载标准值（1.0kN/m）
- L：栏杆长度

## 安全监测计算

### 1. 基坑监测

#### 1.1 报警值确定
| 监测项目 | 累计值 | 变化速率 |
|---------|-------|----------|
| 水平位移 | 30mm（0.3%H） | 3mm/d |
| 沉降 | 20mm | 2mm/d |
| 锚索拉力 | 80%设计值 | - |

### 2. 塔吊安全

#### 2.1 塔吊基础计算
- 地基承载力：P ≤ f
- 基础配筋：按计算确定
- 稳定性：抗倾覆系数 ≥ 1.5

## 注意事项

### 计算要点
1. 事故统计要实事求是
2. 隐患整改要有时间节点
3. 监测数据要及时分析

### 常见问题
1. 统计口径不一致
2. 隐患分类不准确
3. 整改期限不合理

### 改进建议
- 统一统计标准
- 规范隐患分类
- 合理设定整改期限

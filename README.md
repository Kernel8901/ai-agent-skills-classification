## 🚀 Quick Start

### Browse Skills Online
Simply navigate through the directory structure on GitHub to explore skills by category.

### Clone for Local Use
```bash
git clone https://github.com/Kernel8901/ai-agent-skills-classification.git
cd ai-agent-skills-classification
```

### Find Specific Skills
```bash
# Search for React-related skills
find . -name "*.md" -path "*/网页与前端开发/*" -exec grep -l "react" {} \;

# List all AI/LLM skills
ls "人工智能与大型语言模型/"
```

### Integrate into Your Project
Each skill folder contains:
- `SKILL.md` - Detailed skill description and usage examples
- `_meta.json` - Structured metadata (name, tags, version, etc.)
- Additional resources as needed

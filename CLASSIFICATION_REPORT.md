## 📖 使用建议

1. **浏览技能库**
   ```bash
   # 查看某个类别的技能列表
   ls "人工智能与大型语言模型/" | head -20
   
   # 查看所有类别
   ls -d */
   ```

2. **查找特定技能**
   ```bash
   # 在所有技能中搜索关键词
   grep -r "react" . --include="SKILL.md"
   
   # 在特定类别中搜索
   grep -r "python" "网页与前端开发/" --include="SKILL.md"
   ```

3. **集成到项目**
   - 直接引用本仓库作为技能索引
   - 通过 Git submodule 引入到您的项目
   - 复制所需类别的技能到您的 Agent 系统

4. **贡献改进**
   - 发现分类不当的技能？提交 Issue 或 PR
   - 有新技能要添加？按照现有格式创建新文件夹
   - 优化元数据？编辑 `_meta.json` 文件
